## Key Design Decisions

1. **S-bend waypoints** `[-2,0] → [0,0] → [0.5,0.5] → [2,0.5]` — shallow curve avoids item-tip instability while demonstrating non-trivial path change.
2. **Zone partition** used as bookkeeping tag (`robots=['lane_a_zone','lane_b_zone']`); actual routing is driven by semantic label read at diverge proximity sensor.
3. **Two divert conveyors** angled off track end (`surface_velocity=[0.20, ±0.20, 0]`) replace a robot arm for lane separation — keeps the template arm-free.
4. **CPU physics** (`enable_gpu_dynamics=False`, MBP broadphase) for determinism on small 6-item scene.
