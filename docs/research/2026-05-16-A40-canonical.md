## Key design choices

1. **set_semantic_label uses `class_name=label`** (not `label=`) — avoids TC_UNKNOWN_KWARG WARN present in base kit-prep-operator template.
2. **Two-phase code**: Phase 1 is vanilla `setup_pick_place_controller`; Phase 2 adds `add_vision_classifier_gate` + `setup_pick_place_with_vision` for conditional re-pick. Keeps the canonical composable: Phase 1 alone passes functional gate.
3. **Camera geometry**: nadir at z=1.60 over tray at z=0.775 (0.825m above parts). focal_length=18mm covers full 0.55m tray width at that height.
4. **No validate_annotations**: backlog description listed it, but it validates SDG annotation files — not applicable here. Replaced with `add_vision_classifier_gate` which is the correct runtime completeness tool.
