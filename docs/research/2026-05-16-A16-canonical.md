## Canonical Design

**Category:** yrkesroll  
**Pattern:** train

### 3-Stage DR Curriculum Flow

```
Stage 1 (fixed)   → no DR  → baseline scene
Stage 2 (photometric) → DomeLight intensity + cube color randomized
                      → add_domain_randomizer ×2 + analyze_randomization(30)
Stage 3 (full)    → Stage 2 + texture + pose jitter
                      → add_domain_randomizer ×2 more + analyze_randomization(30)
                      → preview_dr(9 samples) → visual gate
Training          → suggest_dr_ranges → apply_dr_preset('indoor_industrial')
                  → create_isaaclab_env → clone_envs(16) → launch_training(PPO,20k)
                  → checkpoint_training(tag='stage3_baseline')
```

### Distinctive Tool Coverage

First canonical to use `suggest_dr_ranges + apply_dr_preset` together. Unique multi-stage
DR authoring loop (4× `add_domain_randomizer`). Contrast with existing train canonicals:

- `CP-NEW-parallel-env-scaling-32` — throughput benchmarking, fixed env
- `CP-NEW-eureka-pick-place-reward` — reward iteration loop
- **this** — DR distribution management, progressive curriculum
