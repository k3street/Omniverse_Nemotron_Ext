# tool_executor.py recovered-state forensics — 2026-05-13

Phase 13 archive — the lines 32-400 "Recovered state for bundled PR
handlers" block from the original 35,842-line `tool_executor.py` has
been fully drained. This document records what was there for future
reference; the live code under `handlers/` is the source of truth.

## Origin

The "Recovered state" block was inherited from a local-QA branch that
restored 182 broken module-level references after a bulk extraction
script missed them. Each symbol was originally tied to a feature
branch (named in the inline `# from: feat/...` comments).

## Migration history

Phase 8 (29 waves, 2026-05-13) systematically moved every symbol out
of the block into one of:

| Destination | Symbol kind | Examples |
|---|---|---|
| `handlers/<theme>.py` module body | theme-local constants/functions | `_ARENA_SCENE_MAP`, `_OG_TEMPLATES`, `_ROBOT_FIX_PROFILES` |
| `handlers/_shared.py` | cross-theme constants/functions | `_SAFE_XFORM_SNIPPET`, `_OG_NODE_TYPE_MAP`, `_probe_gpu_capability` |
| `handlers/_state.py` | mutable state singletons | `ASYNC_TASKS`, `WRITE_LOCK_QUEUE`, `LockedPatch` class |

Per-wave commits in `git log --grep "phase-8"` document each move.

## Final state

After all waves:
- 0 DEAD symbols (none unreferenced)
- 0 INTERNAL_ONLY symbols (none used only inside tool_executor.py)
- 0 HANDLER_USED symbols (none reachable via direct handler import)
- 0 EXTERNAL_USED symbols

`tool_executor.py` retains only:
1. The dispatch core: `execute_tool_call`, `_apply_result_cap`.
2. `register_handlers(DATA_HANDLERS, CODE_GEN_HANDLERS)` — Phase 9
   dispatch swap.
3. Five back-compat alias imports from `handlers/_state.py` so any
   stray `_te.X` callsite continues to work:
   - `_LockedPatch` → `_state.LockedPatch`
   - `_StageWriteLockQueue` → `_state.StageWriteLockQueue`
   - `_WRITE_LOCK_QUEUE` → `_state.WRITE_LOCK_QUEUE`
   - `_ASYNC_TASKS` → `_state.ASYNC_TASKS`
   - `_ASYNC_TASKS_LOCK` → `_state.ASYNC_TASKS_LOCK`
4. Migration-marker comments (one per moved symbol) preserving the
   audit trail of which symbol went where in which wave.
