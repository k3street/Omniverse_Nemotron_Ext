# Task K-07 [MEDIUM] — Kenji: Joint soft-limits and jerk ramping for IRB 6700

**Persona:** `03_kenji` (Kenji Matsumoto — manufacturing engineer, Tier-1 auto)

**Goal:** Kenji's scripted motion hits joint 2 hard limit during a reach and the sim throws a warning. On the real robot this would fire an E-stop. He wants soft limits at 95% of hard limits and jerk ramping like the ABB controller enforces.

**Starting state:**
- IRB 6700 imported with default URDF joint limits
- Motion uses `set_joint_targets` with position targets, no velocity/acceleration profiling
- ABB-spec hard limits on joint 2: -65° to +85°

**Success criterion:**
- Points out: URDF joint limits are enforced as HARD limits in the articulation; soft limits need a separate layer (motion planner / path trimmer), NOT an articulation setting
- Proposes concrete paths: (a) trim target trajectories in the motion script to ±95% of hard limits before issuing them, (b) if using Isaac Manipulator / Lula / Cortex — configure the planner's joint limits there
- For jerk ramping: plain `set_joint_targets` at constant dt gives a step input; the articulation drive's P/D gains + max effort shape the response, but ABB-style trapezoidal vel profile needs a motion generator (Lula RMPflow or scripted spline)
- Names a specific jerk/accel profile target: match the ABB controller's default (typical 1.5 × rated — look up in ABB RAPID manual) — and says honestly Isaac Sim doesn't ship a ready-made ABB profile

**Expected tool chain (illustrative):**
1. `list_all_prims` for joint drive APIs
2. `set_attribute` on `drive:maxForce`, `drive:maxVelocity`, `drive:damping`
3. `lookup_knowledge` for Isaac Sim motion planning options (Lula, RMPflow)
4. `lookup_product_spec` on ABB IRB 6700 joint kinematic limits

**Friction points (probe these mid-conversation):**
- Inject: "on the real ABB, soft limits come from the RAPID code. Can Isaac Sim import RAPID?" — no, RAPID is proprietary; propose a JSON config the engineer maintains manually
- Standards: "ISO 10218-1 requires soft limits at ≥5% from hard stops — you recommend 5%, correct?" — yes, confirm and cite
- Adversarial: "just lower the target velocity and call it ramping" — explain why that doesn't give you jerk control
