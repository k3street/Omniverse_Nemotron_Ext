# Task CW-48 [EASY] — Save the stage to disk

**Persona:** common workflow

**Goal:** Save the current stage to `/tmp/cw48_stage.usd`.

**Starting state:**
- A stage with a few prims

**Success criterion:**
- Calls `save_stage(path="/tmp/cw48_stage.usd")` or `export_stage(...)`
- Reply confirms the file was written

**Expected tool chain:**
1. `save_stage(path="/tmp/cw48_stage.usd")` OR `export_stage(path="/tmp/cw48_stage.usd")`

**Time budget:** 1 minute.

## Pre-session setup

```python
import omni.usd
from pxr import UsdGeom, UsdLux
ctx = omni.usd.get_context()
stage = ctx.get_stage()
UsdGeom.Xform.Define(stage, "/World")
UsdLux.DomeLight.Define(stage, "/World/DomeLight")
UsdGeom.Cube.Define(stage, "/World/Cube")
UsdGeom.Sphere.Define(stage, "/World/Sphere")
```
