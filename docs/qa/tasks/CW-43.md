# Task CW-43 [EASY] — Measure distance between two prims

**Persona:** common workflow

**Goal:** Measure the distance between `/World/A` and `/World/B`.

**Starting state:**
- Cube A at (0, 0, 0), cube B at (3, 4, 0) — expected distance = 5

**Success criterion:**
- Calls `measure_distance(prim_a="/World/A", prim_b="/World/B")` or equivalent
- Reply reports a distance roughly equal to 5

**Expected tool chain:**
1. `measure_distance(prim_a="/World/A", prim_b="/World/B")`

**Time budget:** 1 minute.

## Pre-session setup

```python
import omni.usd
from pxr import UsdGeom, UsdLux, Gf
ctx = omni.usd.get_context()
stage = ctx.get_stage()
UsdGeom.Xform.Define(stage, "/World")
UsdLux.DomeLight.Define(stage, "/World/DomeLight")
a = UsdGeom.Cube.Define(stage, "/World/A").GetPrim()
UsdGeom.Xformable(a).AddTranslateOp().Set(Gf.Vec3d(0, 0, 0))
b = UsdGeom.Cube.Define(stage, "/World/B").GetPrim()
UsdGeom.Xformable(b).AddTranslateOp().Set(Gf.Vec3d(3, 4, 0))
```
