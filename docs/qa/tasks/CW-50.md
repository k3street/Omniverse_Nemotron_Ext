# Task CW-50 [EASY] — Teleport a prim to a new pose

**Persona:** common workflow

**Goal:** Move `/World/Cube` to position (2, 0, 1).

**Starting state:**
- Cube at `/World/Cube` at origin

**Success criterion:**
- Calls `teleport_prim(prim_path="/World/Cube", position=[2, 0, 1])` or
  `set_attribute` on translate
- Reply confirms the new position

**Expected tool chain:**
1. `teleport_prim(prim_path="/World/Cube", position=[2, 0, 1])`

**Time budget:** 1 minute.

## Pre-session setup

```python
import omni.usd
from pxr import UsdGeom, UsdLux, Gf
ctx = omni.usd.get_context()
stage = ctx.get_stage()
UsdGeom.Xform.Define(stage, "/World")
UsdLux.DomeLight.Define(stage, "/World/DomeLight")
cube = UsdGeom.Cube.Define(stage, "/World/Cube").GetPrim()
UsdGeom.Xformable(cube).AddTranslateOp().Set(Gf.Vec3d(0, 0, 0))
```
