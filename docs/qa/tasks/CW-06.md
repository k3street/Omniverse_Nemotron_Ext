# Task CW-06 [EASY] — Apply collision API to all cubes in batch

**Persona:** common workflow

**Goal:** The scene has 5 cubes. Apply PhysicsCollisionAPI to all of them in one batch operation.

**Starting state:**
- 5 Cube prims at /World/Cube_0..4 with no physics schemas
- Default light + physics scene

**Success criterion:**
- All 5 cubes have PhysicsCollisionAPI applied afterward
- Uses a batch operation (`bulk_apply_schema` or equivalent), NOT 5 separate calls
- Reply confirms which prims received the schema

**Expected tool chain:**
1. `find_prims_by_schema` or `find_prims_by_name(pattern="Cube_*")` to discover targets
2. `bulk_apply_schema(prim_paths=[...], schema_name="PhysicsCollisionAPI")`
3. `list_applied_schemas` to verify on at least one cube

**Time budget:** 2 minutes.

## Pre-session setup

```python
import omni.usd
from pxr import UsdGeom, UsdPhysics, UsdLux, Gf
ctx = omni.usd.get_context()
stage = ctx.get_stage()
UsdGeom.Xform.Define(stage, "/World")
UsdPhysics.Scene.Define(stage, "/World/PhysicsScene")
UsdLux.DomeLight.Define(stage, "/World/DomeLight")
for i in range(5):
    p = UsdGeom.Cube.Define(stage, f"/World/Cube_{i}").GetPrim()
    UsdGeom.Xformable(p).AddTranslateOp().Set(Gf.Vec3d(i*0.3, 0, 0.5))
    UsdGeom.Xformable(p).AddScaleOp().Set(Gf.Vec3d(0.05, 0.05, 0.05))
```
