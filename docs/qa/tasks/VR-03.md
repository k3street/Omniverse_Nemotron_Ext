# Task VR-03 [EASY] — Resolver pilot #2: deictic reference (ambiguous)

**Persona:** common workflow

**Goal:** Move the cube to origin.

**Starting state:**
- Stage with three cubes at /World/Cube_1, /World/Cube_2, /World/Cube_3 at distinct positions

**Success criterion:**
- Agent calls `resolve_prim_reference` and gets ambiguous=true with 3 candidates
- Agent does NOT pick one arbitrarily — must ask the user "which one?"
- Reply lists at least two of the three candidate paths
- No teleport_prim is issued (since we never resolved the ambiguity)

**Expected tool chain:**
1. `resolve_prim_reference(name_hint="cube")` → returns ambiguous list of 3
2. Agent asks user via reply text — no further tool calls

**Time budget:** 1 minute, 1 turn.

## Pre-session setup

```python
import omni.usd
from pxr import UsdGeom, UsdLux, Gf
ctx = omni.usd.get_context()
stage = ctx.get_stage()
UsdGeom.Xform.Define(stage, "/World")
UsdLux.DomeLight.Define(stage, "/World/DomeLight")
for i in range(1, 4):
    c = UsdGeom.Cube.Define(stage, f"/World/Cube_{i}").GetPrim()
    UsdGeom.Xformable(c).AddTranslateOp().Set(Gf.Vec3d(i*0.5, 0, 0))
```
