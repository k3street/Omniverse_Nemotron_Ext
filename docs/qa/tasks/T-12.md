## Pre-session setup

Harness-only: seed a rigid body with `physics:mass = 0` so PhysX will NaN on Play. Assist must diagnose from scene state alone; do NOT mention this fixture exists.

```python
import omni.usd
from pxr import UsdGeom, UsdPhysics
ctx = omni.usd.get_context()
stage = ctx.get_stage()
UsdGeom.Xform.Define(stage, "/World")
UsdPhysics.Scene.Define(stage, "/World/PhysicsScene")
UsdGeom.Xform.Define(stage, "/World/Ground")
cube = UsdGeom.Cube.Define(stage, "/World/BrokenCube")
UsdPhysics.RigidBodyAPI.Apply(cube.GetPrim())
UsdPhysics.CollisionAPI.Apply(cube.GetPrim())
mass_api = UsdPhysics.MassAPI.Apply(cube.GetPrim())
mass_api.GetMassAttr().Set(0.0)   # THE FAULT — zero-mass rigid body
print("seeded T-12 zero_mass fault on /World/BrokenCube")
```
