# Task CW-24 [MEDIUM] — Check if two prims overlap geometrically

**Persona:** common workflow

**Goal:** Determine whether the two cubes /World/CubeA and /World/CubeB overlap. Report yes/no with reasoning.

**Starting state:**
- /World/CubeA at (0,0,0) size 0.2m
- /World/CubeB at (0.05, 0, 0) size 0.2m (overlapping by 0.15m on X)
- Default light

**Success criterion:**
- Calls `check_collisions` OR `overlap_box` OR queries via `get_bounding_box` on both
- Reply concludes the cubes DO overlap (correct answer)
- Does NOT just guess — uses tools

**Expected tool chain:**
1. `get_bounding_box` on each + compute manually OR
2. `check_collisions` if available
3. Reply with yes/no + bounding-box-based reasoning

**Time budget:** 2 minutes.

## Pre-session setup

```python
import omni.usd
from pxr import UsdGeom, UsdLux, Gf
ctx = omni.usd.get_context()
stage = ctx.get_stage()
UsdGeom.Xform.Define(stage, "/World")
UsdLux.DomeLight.Define(stage, "/World/DomeLight")
a = UsdGeom.Cube.Define(stage, "/World/CubeA").GetPrim()
UsdGeom.Xformable(a).AddTranslateOp().Set(Gf.Vec3d(0, 0, 0))
UsdGeom.Xformable(a).AddScaleOp().Set(Gf.Vec3d(0.1, 0.1, 0.1))
b = UsdGeom.Cube.Define(stage, "/World/CubeB").GetPrim()
UsdGeom.Xformable(b).AddTranslateOp().Set(Gf.Vec3d(0.05, 0, 0))
UsdGeom.Xformable(b).AddScaleOp().Set(Gf.Vec3d(0.1, 0.1, 0.1))
```
