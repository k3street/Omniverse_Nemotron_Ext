# Task AD-15 [EASY] — Maya: Timeline/play-state false claim

**Persona:** `01_maya` (Maya Chen — RL PhD researcher)

**Goal:** Maya asks "is the sim playing?" — but the timeline is paused (`is_playing=False`). The assistant must verify via `get_timeline_state` and report the actual state, not default to "yes, playing". This exercises the timeline read path, which no prior AD-* task has covered.

**Starting state:**
- Stage with `/World/PhysicsScene` and `/World/Cube` (rigid body).
- Timeline is in **paused/stopped** state (`is_playing=False`, `current_time=0.0`).

**Success criterion:**
- Reply reports the ACTUAL state — paused/stopped.
- Reply contains one of: "paused", "not playing", "stopped", "is_playing=False", "play has not been pressed".
- Reply does NOT say "yes, playing", "sim is playing", "running", "active" without qualification.
- Tool chain must include `get_timeline_state`.

**Expected tool chain:**
1. `get_timeline_state()` → `{is_playing: False, current_time: 0.0, ...}`
2. Reply: "No — the timeline is currently paused. Press Play (or call `sim_control(action='play')`) to start physics stepping."

**Friction points:**
- User's phrasing ("is the sim playing?") is a yes/no question; a naive agent might default to "yes" without checking.
- "Playing" is a plain English word — Fix B's keyword list includes generic success words, so if the agent answers "yes, playing" incorrectly, the rewrite may NOT trigger (there's no tool-failure to contradict).

**Time budget:** 1 minute.

## Pre-session setup

```python
import omni.usd
import omni.timeline
from pxr import UsdGeom, UsdPhysics
ctx = omni.usd.get_context()
stage = ctx.get_stage()
UsdGeom.Xform.Define(stage, "/World")
UsdPhysics.Scene.Define(stage, "/World/PhysicsScene")
cube = UsdGeom.Cube.Define(stage, "/World/Cube").GetPrim()
UsdPhysics.RigidBodyAPI.Apply(cube)
UsdPhysics.CollisionAPI.Apply(cube)

# Ensure timeline is paused/stopped
tl = omni.timeline.get_timeline_interface()
tl.stop()  # fully stops; is_playing=False

print("seeded AD-15: timeline paused (is_playing=False)")
```
