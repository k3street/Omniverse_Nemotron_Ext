# Task CW-32 [EASY] — List all lights in the stage

**Persona:** common workflow

**Goal:** List all light prims in the stage.

**Starting state:**
- Three lights: `/World/DomeLight` (DomeLight), `/World/Spot` (SphereLight), `/World/Rect` (RectLight)

**Success criterion:**
- Calls `list_lights()`
- Reply contains all three light paths or names

**Expected tool chain:**
1. `list_lights()`

**Time budget:** 1 minute.

## Pre-session setup

```python
import omni.usd
from pxr import UsdGeom, UsdLux
ctx = omni.usd.get_context()
stage = ctx.get_stage()
UsdGeom.Xform.Define(stage, "/World")
UsdLux.DomeLight.Define(stage, "/World/DomeLight")
UsdLux.SphereLight.Define(stage, "/World/Spot")
UsdLux.RectLight.Define(stage, "/World/Rect")
```
