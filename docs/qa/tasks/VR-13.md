# Task VR-13 [HARD] — Quantitative: vertical offset

**Persona:** common workflow

**Goal:** Place a robot 40cm above the cube.

**Starting state:**
- Stage with /World/Cube (size=0.5m) at origin, default light + physics

**Success criterion:**
- Agent calls `resolve_constraint_phrase(phrase="40cm above")` and gets parsed.value=0.4
- A robot articulation imported into the scene
- The robot's world-space center z is between 0.6m and 0.7m (cube top at z=0.25 + 0.4 ± tolerance)
- Reply confirms the placement

**Vertical sibling of VR-07.** The agent must combine the constraint
(0.4m) with the cube's geometric top (z=0.25 for a 0.5m cube at origin)
to compute the robot's z. Tests whether constraint-fulfillment generalises
beyond just lateral placement.

**Time budget:** 3 minutes.

## Pre-session setup

```python
import omni.usd
from pxr import UsdGeom, UsdLux, UsdPhysics
ctx = omni.usd.get_context()
stage = ctx.get_stage()
UsdGeom.Xform.Define(stage, "/World")
UsdLux.DomeLight.Define(stage, "/World/DomeLight")
UsdPhysics.Scene.Define(stage, "/World/PhysicsScene")
c = UsdGeom.Cube.Define(stage, "/World/Cube").GetPrim()
c.GetAttribute("size").Set(0.5)
```
