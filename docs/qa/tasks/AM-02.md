## Pre-session setup

```python
import omni.usd, omni.timeline
from pxr import UsdGeom, UsdPhysics, Gf
ctx = omni.usd.get_context()
stage = ctx.get_stage()
UsdGeom.Xform.Define(stage, "/World")
UsdPhysics.Scene.Define(stage, "/World/PhysicsScene")

# Robot loaded but with NO drives configured — sits motionless on Play.
robot = UsdGeom.Xform.Define(stage, "/World/Robot").GetPrim()
UsdPhysics.ArticulationRootAPI.Apply(robot)
base = UsdGeom.Cube.Define(stage, "/World/Robot/base").GetPrim()
UsdPhysics.RigidBodyAPI.Apply(base)
UsdPhysics.CollisionAPI.Apply(base)
arm = UsdGeom.Cube.Define(stage, "/World/Robot/arm_link").GetPrim()
UsdGeom.Xformable(arm).AddTranslateOp().Set(Gf.Vec3d(0, 0, 0.3))
UsdPhysics.RigidBodyAPI.Apply(arm)
UsdPhysics.CollisionAPI.Apply(arm)

# Floor for the robot to stand on
floor = UsdGeom.Cube.Define(stage, "/World/Floor").GetPrim()
UsdGeom.Xformable(floor).AddTranslateOp().Set(Gf.Vec3d(0, 0, -0.5))
UsdGeom.Xformable(floor).AddScaleOp().Set(Gf.Vec3d(5, 5, 0.05))
UsdPhysics.CollisionAPI.Apply(floor)

# Press Play — Amir already did this and is staring at a non-moving robot.
omni.timeline.get_timeline_interface().play()
```
