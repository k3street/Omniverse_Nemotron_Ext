# Task CW-29 [HARD] — Generate occupancy map for navigation

**Persona:** common workflow

**Goal:** Generate a 2D occupancy map of the scene at z=0.05, 0.05 m/cell, save to /tmp/occupancy_map.

**Starting state:**
- Scene with ground + 4 obstacle cubes scattered
- Default light + physics scene

**Success criterion:**
- Calls `generate_occupancy_map` with the right Z-slice and resolution
- Specifies output path /tmp/occupancy_map
- Reply confirms map generation (or honestly reports if Nav2 deps missing)

**Expected tool chain:**
1. `generate_occupancy_map(z_slice=0.05, resolution=0.05, output_path="/tmp/occupancy_map", ...)`

**Time budget:** 5 minutes.

## Pre-session setup

```python
import omni.usd
from pxr import UsdGeom, UsdPhysics, UsdLux, Gf
import random
ctx = omni.usd.get_context()
stage = ctx.get_stage()
UsdGeom.Xform.Define(stage, "/World")
UsdPhysics.Scene.Define(stage, "/World/PhysicsScene")
UsdLux.DomeLight.Define(stage, "/World/DomeLight")
ground = UsdGeom.Cube.Define(stage, "/World/Ground").GetPrim()
UsdGeom.Xformable(ground).AddTranslateOp().Set(Gf.Vec3d(0, 0, -0.05))
UsdGeom.Xformable(ground).AddScaleOp().Set(Gf.Vec3d(5, 5, 0.05))
UsdPhysics.CollisionAPI.Apply(ground)
random.seed(42)
for i in range(4):
    p = UsdGeom.Cube.Define(stage, f"/World/Obstacle_{i}").GetPrim()
    UsdGeom.Xformable(p).AddTranslateOp().Set(Gf.Vec3d(random.uniform(-2,2), random.uniform(-2,2), 0.2))
    UsdGeom.Xformable(p).AddScaleOp().Set(Gf.Vec3d(0.2, 0.2, 0.2))
    UsdPhysics.CollisionAPI.Apply(p)
```
