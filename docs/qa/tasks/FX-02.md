## Pre-session setup

```python
import omni.usd
from pxr import UsdGeom, Gf
ctx = omni.usd.get_context()
stage = ctx.get_stage()

cam = UsdGeom.Camera.Define(stage, "/World/Camera")
xf = UsdGeom.Xformable(cam)
xf.AddTranslateOp().Set(Gf.Vec3d(0.0, -3.0, 1.5))
```
