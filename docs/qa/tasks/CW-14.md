# Task CW-14 [HARD] — Setup ROS2 bridge for joint state publishing

**Persona:** common workflow

**Goal:** Configure the ROS2 bridge so that the Franka articulation publishes its joint state on the topic `/joint_states`.

**Starting state:**
- Real Franka at /World/Robot
- Default light + physics scene

**Success criterion:**
- Calls `setup_ros2_bridge` (or `configure_ros2_bridge`) for the robot
- Configures joint state publishing for /World/Robot
- Reply confirms the bridge is configured (acknowledges if AMENT_PREFIX_PATH unsourced is a known caveat)

**Expected tool chain:**
1. `setup_ros2_bridge(robot_path="/World/Robot", profile="joint_state")` OR
   `configure_ros2_bridge(sensors=["joint_state"], robot_path="/World/Robot")`
2. Reply with status

**Time budget:** 5 minutes.

## Pre-session setup

```python
import omni.usd
from pxr import UsdGeom, UsdPhysics, Gf, UsdLux
ctx = omni.usd.get_context()
stage = ctx.get_stage()
UsdGeom.Xform.Define(stage, "/World")
UsdPhysics.Scene.Define(stage, "/World/PhysicsScene")
UsdLux.DomeLight.Define(stage, "/World/DomeLight")

# Manually-authored Franka-like articulation
robot = UsdGeom.Xform.Define(stage, "/World/Robot").GetPrim()
UsdPhysics.ArticulationRootAPI.Apply(robot)
UsdPhysics.RigidBodyAPI.Apply(robot)
UsdPhysics.MassAPI.Apply(robot).CreateMassAttr(15.0)

base = UsdGeom.Cube.Define(stage, "/World/Robot/base").GetPrim()
UsdPhysics.RigidBodyAPI.Apply(base)
UsdPhysics.CollisionAPI.Apply(base)

# A few joints so /joint_states would carry meaningful data
for i in range(1, 4):
    link = UsdGeom.Cube.Define(stage, f"/World/Robot/link{i}").GetPrim()
    UsdGeom.Xformable(link).AddTranslateOp().Set(Gf.Vec3d(0, 0, 0.1*i))
    UsdGeom.Xformable(link).AddScaleOp().Set(Gf.Vec3d(0.05, 0.05, 0.05))
    UsdPhysics.RigidBodyAPI.Apply(link)
    UsdPhysics.CollisionAPI.Apply(link)
    joint = UsdPhysics.RevoluteJoint.Define(stage, f"/World/Robot/panda_joint{i}")
    parent = "base" if i == 1 else f"link{i-1}"
    joint.CreateBody0Rel().SetTargets([f"/World/Robot/{parent}"])
    joint.CreateBody1Rel().SetTargets([f"/World/Robot/link{i}"])
    joint.CreateAxisAttr("Z")
    drive = UsdPhysics.DriveAPI.Apply(joint.GetPrim(), "angular")
    drive.CreateTypeAttr("force")

```
