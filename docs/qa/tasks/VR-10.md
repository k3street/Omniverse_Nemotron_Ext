# Task VR-10 [HARD] — Filter-deictic: "the SMALL cube" with mixed sizes

**Persona:** common workflow

**Goal:** Move the small cube to (1, 0, 0).

**Starting state:**
- Three cubes at /World/Cube_small (size=0.05), /World/Cube_med (size=0.2), /World/Cube_large (size=0.5), all at distinct origin-line positions

**Success criterion:**
- Only the small cube is moved — its world position ends at (1, 0, 0) ± 1mm
- The medium and large cubes are NOT moved (their positions remain at the pre-session-setup values)
- Reply confirms the move and identifies the small cube specifically

**Expected tool chain:**
- `resolve_prim_reference(name_hint="cube")` returns 3 candidates
- Agent must distinguish them by size — typically by:
    - Calling `get_bounding_box` on each candidate
    - OR reading the names (`Cube_small` is a hint, but agent shouldn't rely on names alone)
    - OR asking the user for clarification (also acceptable, if user gets the choice presented sensibly)
- Then `teleport_prim(prim_path="/World/Cube_small", position=[1,0,0])`

**This probes a real gap.** resolve_prim_reference returns candidates with
prim_path/type/position, but NO size data. Agent must either follow up with
a per-candidate bbox query or fall back to name-based heuristics — neither
is great. Real failure mode: agent picks an arbitrary cube.

**Time budget:** 2 minutes.

## Pre-session setup

```python
import omni.usd
from pxr import UsdGeom, UsdLux, Gf
ctx = omni.usd.get_context()
stage = ctx.get_stage()
UsdGeom.Xform.Define(stage, "/World")
UsdLux.DomeLight.Define(stage, "/World/DomeLight")

c_small = UsdGeom.Cube.Define(stage, "/World/Cube_small").GetPrim()
c_small.GetAttribute("size").Set(0.05)
UsdGeom.Xformable(c_small).AddTranslateOp().Set(Gf.Vec3d(0, 0, 0))

c_med = UsdGeom.Cube.Define(stage, "/World/Cube_med").GetPrim()
c_med.GetAttribute("size").Set(0.2)
UsdGeom.Xformable(c_med).AddTranslateOp().Set(Gf.Vec3d(0.5, 0, 0))

c_large = UsdGeom.Cube.Define(stage, "/World/Cube_large").GetPrim()
c_large.GetAttribute("size").Set(0.5)
UsdGeom.Xformable(c_large).AddTranslateOp().Set(Gf.Vec3d(1.5, 0, 0))
```
