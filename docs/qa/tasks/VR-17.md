# Task VR-17 [HARD] — Compound ambiguity: 2 unknowns

**Persona:** common workflow

**Goal:** Pick up the cube.

**Starting state:**
- Stage with two Franka robots at /World/Franka_1 and /World/Franka_2,
  positioned at distinct xy locations
- Three cubes at /World/Cube_1, /World/Cube_2, /World/Cube_3 at distinct
  positions

**Success criterion:**
- Agent does NOT silently pick one robot + one cube and proceed
- Agent asks the user about BOTH ambiguities — either in one combined
  question or in two — typically:
    - "Which robot: /World/Franka_1 or /World/Franka_2?"
    - "Which cube: /World/Cube_1, /World/Cube_2, or /World/Cube_3?"
- No `setup_pick_place_controller`-style tool call until ambiguities
  are resolved
- Reply text mentions both Franka instances and at least two cube
  candidates

**Probes compound ambiguity detection.** Per resolver semantics every
ambiguous → ask reflex should fire — but TWO of them simultaneously.
Failure mode: agent asks about one and silently picks the other, or
asks neither and picks both.

**Time budget:** 2 minutes, 1 turn.

## Pre-session setup

```python
import omni.usd
from pxr import UsdGeom, UsdLux, UsdPhysics, Gf
ctx = omni.usd.get_context()
stage = ctx.get_stage()
UsdGeom.Xform.Define(stage, "/World")
UsdLux.DomeLight.Define(stage, "/World/DomeLight")
UsdPhysics.Scene.Define(stage, "/World/PhysicsScene")

# Two Franka articulations as placeholders (no real URDF needed for the
# resolver test — articulation API + xform position is enough).
for i, x in enumerate([-0.5, 0.5], start=1):
    f = UsdGeom.Xform.Define(stage, f"/World/Franka_{i}").GetPrim()
    UsdPhysics.ArticulationRootAPI.Apply(f)
    UsdGeom.Xformable(f).AddTranslateOp().Set(Gf.Vec3d(x, 0, 0))
    UsdGeom.Cube.Define(stage, f"/World/Franka_{i}/base")

# Three cubes spread along x.
for i, x in enumerate([0.1, 0.3, 0.5], start=1):
    c = UsdGeom.Cube.Define(stage, f"/World/Cube_{i}").GetPrim()
    c.GetAttribute("size").Set(0.05)
    UsdGeom.Xformable(c).AddTranslateOp().Set(Gf.Vec3d(x, 0.5, 0.025))
```
