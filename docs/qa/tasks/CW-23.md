# Task CW-23 [EASY] — Export the current stage to a USD file

**Persona:** common workflow

**Goal:** Save the current stage to /tmp/exported_stage.usda.

**Starting state:**
- Stage with a few prims
- Default light

**Success criterion:**
- Calls `save_stage` or `export_stage` with the target path
- Reply confirms the file was written
- The file actually exists on disk afterward (judge can check)

**Expected tool chain:**
1. `save_stage(file_path="/tmp/exported_stage.usda")` OR `export_stage(...)`

**Time budget:** 1 minute.

## Pre-session setup

```python
import omni.usd
from pxr import UsdGeom, UsdLux, Gf
ctx = omni.usd.get_context()
stage = ctx.get_stage()
UsdGeom.Xform.Define(stage, "/World")
UsdLux.DomeLight.Define(stage, "/World/DomeLight")
for i in range(3):
    p = UsdGeom.Cube.Define(stage, f"/World/Cube_{i}").GetPrim()
    UsdGeom.Xformable(p).AddTranslateOp().Set(Gf.Vec3d(i*0.3, 0, 0.1))
    UsdGeom.Xformable(p).AddScaleOp().Set(Gf.Vec3d(0.05, 0.05, 0.05))
```
