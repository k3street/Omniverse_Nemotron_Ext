# Task CW-37 [EASY] — Find all prims with Material schema

**Persona:** common workflow

**Goal:** List all prims of type Material in the stage.

**Starting state:**
- Two materials at `/World/Looks/Mat1` (UsdShade.Material) and `/World/Looks/Mat2`

**Success criterion:**
- Calls `find_prims_by_schema(schema="Material")` or equivalent
- Reply lists both Mat1 and Mat2

**Expected tool chain:**
1. `find_prims_by_schema(schema_name="Material")`

**Time budget:** 1 minute.

## Pre-session setup

```python
import omni.usd
from pxr import UsdGeom, UsdLux, UsdShade
ctx = omni.usd.get_context()
stage = ctx.get_stage()
UsdGeom.Xform.Define(stage, "/World")
UsdLux.DomeLight.Define(stage, "/World/DomeLight")
UsdGeom.Scope.Define(stage, "/World/Looks")
UsdShade.Material.Define(stage, "/World/Looks/Mat1")
UsdShade.Material.Define(stage, "/World/Looks/Mat2")
```
