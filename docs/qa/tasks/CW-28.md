# Task CW-28 [EASY] — Start the simulation

**Persona:** common workflow

**Goal:** Start (Play) the simulation timeline. Confirm timeline state afterwards.

**Starting state:**
- Stage with a robot
- Timeline currently paused

**Success criterion:**
- Calls `sim_control(action="play")` (or equivalent)
- Calls `get_timeline_state` afterward to verify it's playing
- Reply confirms the simulation is now running

**Expected tool chain:**
1. `sim_control(action="play")` 
2. `get_timeline_state` to verify

**Time budget:** 1 minute.

## Pre-session setup

```python
import omni.usd
from pxr import UsdGeom, UsdPhysics, UsdLux
ctx = omni.usd.get_context()
stage = ctx.get_stage()
UsdGeom.Xform.Define(stage, "/World")
UsdPhysics.Scene.Define(stage, "/World/PhysicsScene")
UsdLux.DomeLight.Define(stage, "/World/DomeLight")
robot = UsdGeom.Xform.Define(stage, "/World/Robot").GetPrim()
UsdPhysics.ArticulationRootAPI.Apply(robot)
UsdGeom.Cube.Define(stage, "/World/Robot/base")
```
