# Task VR-18 [HARD] — Multi-stage assembly line

**Persona:** common workflow

**Goal:** Build a 3-station assembly line. A 5cm cube starts in a bin at
station A on the left (x ≈ -3). A Franka picks it up and places it on
the first conveyor running to station B (middle, x ≈ 0). At B a second
Franka picks the cube from the conveyor and places it on a second
conveyor running to station C on the right (x ≈ +3), where the cube
ends up in a final output bin. Build the full layout statically — the
robots and conveyors don't need to actually move yet, just be placed
so the pipeline reads as a coherent assembly line.

**Starting state:**
- Empty stage with default light + physics scene + ground plane

**Success criterion:**
- ≥2 robot articulations exist (PhysicsArticulationRootAPI on each)
  with distinct world-x positions (one in the left half, one in the
  middle half — separation > 1m)
- ≥2 conveyor-like prims (created via `create_conveyor` OR
  long-stretched cube prims with reasonable scale, > 2m in one
  horizontal axis each)
- ≥3 bin / container prims spread across the layout (input at A,
  intermediate at B, output at C)
- ≥1 cube prim sized roughly 0.05m
- Reply describes the pipeline shape and identifies all prim paths
- Total prim-count under /World ≥ 8 (so the agent didn't take a
  shortcut and skip components)

**Expected tool chain (any reasonable order):**
1. `resolve_size_adjective(adjective="small", object_class="cube")` (5cm)
2. `resolve_robot_class(robot_class="manipulator")` ×2 — or directly
   pass robot_name="franka_panda" twice
3. `create_prim(prim_type="Cube", size=0.05, ...)` for the part
4. `create_bin(...)` ×3 for input/intermediate/output bins
5. `create_conveyor(...)` ×2 (or `create_prim` cube-as-conveyor ×2)
6. `robot_wizard(robot_name="franka_panda", dest_path="/World/Franka_1", position=[..., ..., ...])` ×2
7. (Optional) `setup_pick_place_controller` ×2 OR
   `resolve_skill_composition(skill="pick_and_place")` ×2 — useful
   but not strictly required for the static-layout goal
8. (Optional) `place_on_top_of(prim_path="/World/Cube", target_prim_path="/World/InputBin")`

**Probe.** Closer to real industrial workflow than any prior VR task.
Tests compositional capacity: the agent must:
  - Spatially reason about three discrete x positions (A, B, C)
  - Decide robot placements relative to conveyor endpoints
  - Sequence creates such that placement references valid targets
  - Not get lost in the long task description

Failure modes to watch:
  - Skipping components ('built one robot, skipped the other')
  - Z-fighting / misaligned conveyors / robots floating
  - Bin placements that don't sit at conveyor ends

**Time budget:** 5 minutes wall, 4 turns max.
