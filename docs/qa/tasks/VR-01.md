# Task VR-01 [EASY] — Resolver pilot #1+#3: spatial + size

**Persona:** common workflow

**Goal:** Place a small cube at the origin and a tiny sphere on top of it.

**Starting state:**
- Empty stage with default light

**Success criterion:**
- A Cube prim exists at world origin with size ≈ 0.05m
- A Sphere prim exists with radius ≈ 0.02m, with its center above the cube top (z > 0.05)
- Reply confirms both prims and the stacking

**Expected tool chain:**
1. `resolve_size_adjective(adjective="small", object_class="cube")`
2. `resolve_size_adjective(adjective="tiny", object_class="sphere")`
3. `create_prim(prim_type="Cube", size=0.05, ...)`
4. `create_prim(prim_type="Sphere", radius=0.02, ...)`
5. `place_on_top_of(prim_path="/World/Sphere", target_prim_path="/World/Cube")`

**Time budget:** 2 minutes.

## Pre-session setup

```python
import omni.usd
from pxr import UsdGeom, UsdLux
ctx = omni.usd.get_context()
stage = ctx.get_stage()
UsdGeom.Xform.Define(stage, "/World")
UsdLux.DomeLight.Define(stage, "/World/DomeLight")
```
