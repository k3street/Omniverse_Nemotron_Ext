# Task CW-41 [EASY] — Compute volume of a mesh

**Persona:** common workflow

**Goal:** Compute and report the volume of `/World/Cube`.

**Starting state:**
- A unit cube at `/World/Cube` (USD default size = 2)

**Success criterion:**
- Calls `compute_volume(prim_path="/World/Cube")`
- Reply contains a numeric volume (default cube extent is 2 → volume 8)

**Expected tool chain:**
1. `compute_volume(prim_path="/World/Cube")`

**Time budget:** 1 minute.

## Pre-session setup

```python
import omni.usd
from pxr import UsdGeom, UsdLux
ctx = omni.usd.get_context()
stage = ctx.get_stage()
UsdGeom.Xform.Define(stage, "/World")
UsdLux.DomeLight.Define(stage, "/World/DomeLight")
UsdGeom.Cube.Define(stage, "/World/Cube")
```
