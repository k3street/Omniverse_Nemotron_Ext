# Task CW-19 [EASY] — Measure distance between two prims

**Persona:** common workflow

**Goal:** Measure the Euclidean distance between /World/CubeA at (0,0,0) and /World/CubeB at (1, 0, 0). Report the result in meters.

**Starting state:**
- Two cubes at the specified positions
- Default light

**Success criterion:**
- Calls `measure_distance(prim_a="/World/CubeA", prim_b="/World/CubeB")`
- Reply reports value ≈ 1.0 m (within float precision)
- Does NOT compute manually — uses the tool

**Expected tool chain:**
1. `measure_distance(prim_a="/World/CubeA", prim_b="/World/CubeB")`

**Time budget:** 1 minute.

## Pre-session setup

```python
import omni.usd
from pxr import UsdGeom, UsdLux, Gf
ctx = omni.usd.get_context()
stage = ctx.get_stage()
UsdGeom.Xform.Define(stage, "/World")
UsdLux.DomeLight.Define(stage, "/World/DomeLight")
a = UsdGeom.Cube.Define(stage, "/World/CubeA").GetPrim()
UsdGeom.Xformable(a).AddTranslateOp().Set(Gf.Vec3d(0, 0, 0))
UsdGeom.Xformable(a).AddScaleOp().Set(Gf.Vec3d(0.05, 0.05, 0.05))
b = UsdGeom.Cube.Define(stage, "/World/CubeB").GetPrim()
UsdGeom.Xformable(b).AddTranslateOp().Set(Gf.Vec3d(1, 0, 0))
UsdGeom.Xformable(b).AddScaleOp().Set(Gf.Vec3d(0.05, 0.05, 0.05))
```
