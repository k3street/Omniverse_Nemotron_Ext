# Task CW-33 [EASY] — Set light intensity

**Persona:** common workflow

**Goal:** Change the intensity of `/World/DomeLight` to 5000.

**Starting state:**
- A DomeLight at `/World/DomeLight` (default intensity)

**Success criterion:**
- Calls `set_light_intensity(prim_path="/World/DomeLight", intensity=5000)` or equivalent
- Reply confirms intensity was set

**Expected tool chain:**
1. `set_light_intensity(prim_path="/World/DomeLight", intensity=5000)`

**Time budget:** 1 minute.

## Pre-session setup

```python
import omni.usd
from pxr import UsdGeom, UsdLux
ctx = omni.usd.get_context()
stage = ctx.get_stage()
UsdGeom.Xform.Define(stage, "/World")
UsdLux.DomeLight.Define(stage, "/World/DomeLight")
```
