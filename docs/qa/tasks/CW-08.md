# Task CW-08 [MEDIUM] — Apply a red OmniPBR material to a sphere

**Persona:** common workflow

**Goal:** Make the sphere at `/World/Ball` red by applying an OmniPBR material with diffuse color (1.0, 0.0, 0.0).

**Starting state:**
- Sphere at /World/Ball with no material binding
- Default light

**Success criterion:**
- A Material prim is created (typically under /World/Looks/)
- The material is bound to /World/Ball via UsdShade.MaterialBindingAPI
- The diffuse_color attribute is set to (1.0, 0.0, 0.0) ±0.05
- Reply names both the material prim and the bound target

**Expected tool chain:**
1. `create_material(...)` (preferred high-level tool) OR `create_prim` + `assign_material`
2. `assign_material` to bind to /World/Ball

**Time budget:** 3 minutes.

## Pre-session setup

```python
import omni.usd
from pxr import UsdGeom, UsdLux
ctx = omni.usd.get_context()
stage = ctx.get_stage()
UsdGeom.Xform.Define(stage, "/World")
UsdLux.DomeLight.Define(stage, "/World/DomeLight")
ball = UsdGeom.Sphere.Define(stage, "/World/Ball").GetPrim()
UsdGeom.Sphere(ball).GetRadiusAttr().Set(0.5)
```
