# Task VR-14 [HARD] — Clarification on vague: build pick-place cell

**Persona:** common workflow / open-ended

**Goal:** Build a pick-and-place cell.

**Starting state:**
- Empty stage with default light + physics + ground plane

**Success criterion:**
- Agent does NOT build a full cell on turn 1. Either:
    - Agent asks at least one clarifying question (which robot? what
      items? what kind of bin?) — preferred, OR
    - Agent calls `resolve_skill_composition(skill="pick_and_place")`
      and either uses sensible defaults explicitly named in the reply
      (Franka + 3 cubes + bin), OR proceeds to build but flags the
      ambiguity in the reply.
- If agent builds, the cell has at least: 1 robot articulation, 1
  conveyor or table, ≥1 cube, 1 bin.

**Probes the asks-when-missing pattern.** A pick-place prompt with no
robot, no items, no bin specs is genuinely under-specified. Real users
either get asked or get a stub they can iterate on — silently building
a generic cell is the production-pain failure mode.

**Time budget:** 4 minutes, 3 turns max.

## Scripted followups

- "Franka. 3 cubes 5cm. A bin to the left of the robot."
