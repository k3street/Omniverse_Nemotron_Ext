# Task CW-12 [MEDIUM] — Configure SDG pipeline for COCO output

**Persona:** common workflow

**Goal:** Set up an SDG (synthetic data generation) pipeline that writes COCO/YOLO annotations to /tmp/sdg_output for the existing scene.

**Starting state:**
- A scene with one cube at /World/Cube and a camera at /World/Camera
- Default light

**Success criterion:**
- Calls `configure_sdg` AND `configure_coco_yolo_writer` (or equivalent)
- Specifies output directory as /tmp/sdg_output
- Reply confirms the writer is configured (tool returns success)

**Expected tool chain:**
1. `configure_coco_yolo_writer(output_dir="/tmp/sdg_output", ...)`
2. `create_sdg_pipeline` OR `configure_sdg(...)` 
3. Reply mentioning output path + writer

**Time budget:** 3 minutes.

## Pre-session setup

```python
import omni.usd
from pxr import UsdGeom, UsdLux, Gf
ctx = omni.usd.get_context()
stage = ctx.get_stage()
UsdGeom.Xform.Define(stage, "/World")
UsdLux.DomeLight.Define(stage, "/World/DomeLight")
cube = UsdGeom.Cube.Define(stage, "/World/Cube").GetPrim()
UsdGeom.Xformable(cube).AddTranslateOp().Set(Gf.Vec3d(0, 0, 0.5))
cam = UsdGeom.Camera.Define(stage, "/World/Camera").GetPrim()
UsdGeom.Xformable(cam).AddTranslateOp().Set(Gf.Vec3d(2, -2, 1.5))
```
