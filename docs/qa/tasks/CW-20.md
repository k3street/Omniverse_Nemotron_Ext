# Task CW-20 [EASY] — Get the bounding box of a prim

**Persona:** common workflow

**Goal:** Report the bounding box (min and max XYZ) of `/World/Sphere`.

**Starting state:**
- Sphere at /World/Sphere with radius 0.5 at origin

**Success criterion:**
- Calls `get_bounding_box(prim_path="/World/Sphere")`
- Reply reports min ≈ (-0.5, -0.5, -0.5) and max ≈ (0.5, 0.5, 0.5)
- Does NOT fabricate values

**Expected tool chain:**
1. `get_bounding_box(prim_path="/World/Sphere")`

**Time budget:** 1 minute.

## Pre-session setup

```python
import omni.usd
from pxr import UsdGeom, UsdLux
ctx = omni.usd.get_context()
stage = ctx.get_stage()
UsdGeom.Xform.Define(stage, "/World")
UsdLux.DomeLight.Define(stage, "/World/DomeLight")
sphere = UsdGeom.Sphere.Define(stage, "/World/Sphere").GetPrim()
UsdGeom.Sphere(sphere).GetRadiusAttr().Set(0.5)
```
