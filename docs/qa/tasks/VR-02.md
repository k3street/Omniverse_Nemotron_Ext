# Task VR-02 [EASY] — Resolver pilot #2: deictic reference (single match)

**Persona:** common workflow

**Goal:** Move the cube to (1, 0, 0).

**Starting state:**
- Stage with exactly ONE cube at /World/Cube at the origin

**Success criterion:**
- Agent calls `resolve_prim_reference` with name_hint matching "cube"
- Cube ends up at world position (1, 0, 0) (within 1mm)
- Reply confirms the move

**Expected tool chain:**
1. `resolve_prim_reference(name_hint="cube")` → returns exact_match=/World/Cube
2. `teleport_prim(prim_path="/World/Cube", position=[1,0,0])`

**Time budget:** 1 minute.

## Pre-session setup

```python
import omni.usd
from pxr import UsdGeom, UsdLux
ctx = omni.usd.get_context()
stage = ctx.get_stage()
UsdGeom.Xform.Define(stage, "/World")
UsdLux.DomeLight.Define(stage, "/World/DomeLight")
```
