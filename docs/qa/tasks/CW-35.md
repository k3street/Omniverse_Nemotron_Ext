# Task CW-35 [EASY] — Get bounding box of a prim

**Persona:** common workflow

**Goal:** Get the world-space bounding box of `/World/Cube`.

**Starting state:**
- A cube at `/World/Cube` scaled to 0.5

**Success criterion:**
- Calls `get_bounding_box(prim_path="/World/Cube")`
- Reply reports min/max corners or extent (any reasonable form)

**Expected tool chain:**
1. `get_bounding_box(prim_path="/World/Cube")`

**Time budget:** 1 minute.

## Pre-session setup

```python
import omni.usd
from pxr import UsdGeom, UsdLux, Gf
ctx = omni.usd.get_context()
stage = ctx.get_stage()
UsdGeom.Xform.Define(stage, "/World")
UsdLux.DomeLight.Define(stage, "/World/DomeLight")
cube = UsdGeom.Cube.Define(stage, "/World/Cube").GetPrim()
UsdGeom.Xformable(cube).AddScaleOp().Set(Gf.Vec3f(0.5, 0.5, 0.5))
```
