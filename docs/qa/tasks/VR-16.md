# Task VR-16 [HARD] — Geometric composition: 3x3 grid

**Persona:** common workflow

**Goal:** Build a 3x3 grid of cubes, 10cm apart, all on the XY plane at z=0.05.

**Starting state:**
- Empty stage with default light

**Success criterion:**
- Exactly 9 Cube prims exist
- Their xy positions form a 3x3 lattice with 10cm spacing (each cube
  has 3 distinct x-values and 3 distinct y-values, with neighbour
  separations of 0.1m ± 1cm)
- All 9 cubes have the same z (within 1cm)
- Reply summarises 3x3 grid + spacing

**Expected tool chain:**
1. (Optional) `resolve_constraint_phrase(phrase="10cm apart")` → 0.1
2. `create_prim(prim_type="Cube", position=[i*0.1, j*0.1, 0.05])` ×9
   for (i, j) in [0..2] × [0..2]

**Quantitative composition probe.** No single resolver covers grid
layout — agent must compute positions from spacing and grid dimension.
Failure mode: wrong spacing, missing cubes, overlapping cubes.

**Time budget:** 3 minutes.
