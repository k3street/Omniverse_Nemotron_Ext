# Task CW-42 [EASY] — Compute surface area of a mesh

**Persona:** common workflow

**Goal:** Compute and report the surface area of `/World/Sphere`.

**Starting state:**
- A unit sphere at `/World/Sphere` (default radius = 1)

**Success criterion:**
- Calls `compute_surface_area(prim_path="/World/Sphere")`
- Reply contains a numeric area (4πr² ≈ 12.57 for r=1)

**Expected tool chain:**
1. `compute_surface_area(prim_path="/World/Sphere")`

**Time budget:** 1 minute.

## Pre-session setup

```python
import omni.usd
from pxr import UsdGeom, UsdLux
ctx = omni.usd.get_context()
stage = ctx.get_stage()
UsdGeom.Xform.Define(stage, "/World")
UsdLux.DomeLight.Define(stage, "/World/DomeLight")
UsdGeom.Sphere.Define(stage, "/World/Sphere")
```
