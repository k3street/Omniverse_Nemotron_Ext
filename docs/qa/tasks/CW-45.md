# Task CW-45 [EASY] — List variant sets on a prim

**Persona:** common workflow

**Goal:** List the variant sets defined on `/World/Asset`.

**Starting state:**
- An Xform at `/World/Asset` with one variant set "shading_variant" with two variants "red" and "blue"

**Success criterion:**
- Calls `list_variant_sets(prim_path="/World/Asset")`
- Reply contains "shading_variant"

**Expected tool chain:**
1. `list_variant_sets(prim_path="/World/Asset")`

**Time budget:** 1 minute.

## Pre-session setup

```python
import omni.usd
from pxr import UsdGeom, UsdLux
ctx = omni.usd.get_context()
stage = ctx.get_stage()
UsdGeom.Xform.Define(stage, "/World")
UsdLux.DomeLight.Define(stage, "/World/DomeLight")
asset = UsdGeom.Xform.Define(stage, "/World/Asset").GetPrim()
vset = asset.GetVariantSets().AddVariantSet("shading_variant")
vset.AddVariant("red")
vset.AddVariant("blue")
```
