# Task VR-11 [EASY] — Empty-stage recovery

**Persona:** common workflow

**Goal:** Move the cube to origin.

**Starting state:**
- Empty stage — NO cubes anywhere

**Success criterion:**
- Agent calls `resolve_prim_reference(name_hint="cube")` and receives no_match=True
- Agent does NOT hallucinate a prim path or call teleport_prim with a guessed path
- Reply tells the user the scene has no cube and offers to create one (or asks for clarification)
- Stage stays free of new prims unless the user said yes (which they didn't)

**Expected tool chain:**
1. `resolve_prim_reference(name_hint="cube")` → no_match=True
2. (no further tool calls — agent reports back)

**Recovery test.** Probes whether the resolver's no_match signal actually
short-circuits the agent's normal "now move it" reflex.

**Time budget:** 1 minute.

## Pre-session setup

```python
import omni.usd
from pxr import UsdGeom, UsdLux
ctx = omni.usd.get_context()
stage = ctx.get_stage()
UsdGeom.Xform.Define(stage, "/World")
UsdLux.DomeLight.Define(stage, "/World/DomeLight")
```
