# Task CW-16 [MEDIUM] — Apply random colors to all cubes

**Persona:** common workflow

**Goal:** The scene has 5 cubes. Apply a domain randomizer that varies their color uniformly per replicator frame.

**Starting state:**
- 5 cubes /World/Cube_0..4 with default material binding
- Default light + physics scene

**Success criterion:**
- Calls `add_domain_randomizer` (or `apply_dr_preset`) targeting the cubes
- Configures color randomization (uniform RGB)
- Reply confirms which prims are randomized

**Expected tool chain:**
1. `find_prims_by_name(pattern="Cube_*")` to gather targets
2. `add_domain_randomizer(target_paths=[...], randomization_type="color", ...)` OR `apply_dr_preset`

**Time budget:** 3 minutes.

## Pre-session setup

```python
import omni.usd
from pxr import UsdGeom, UsdPhysics, UsdLux, Gf
ctx = omni.usd.get_context()
stage = ctx.get_stage()
UsdGeom.Xform.Define(stage, "/World")
UsdPhysics.Scene.Define(stage, "/World/PhysicsScene")
UsdLux.DomeLight.Define(stage, "/World/DomeLight")
for i in range(5):
    p = UsdGeom.Cube.Define(stage, f"/World/Cube_{i}").GetPrim()
    UsdGeom.Xformable(p).AddTranslateOp().Set(Gf.Vec3d(i*0.3, 0, 0.5))
    UsdGeom.Xformable(p).AddScaleOp().Set(Gf.Vec3d(0.05, 0.05, 0.05))
```
