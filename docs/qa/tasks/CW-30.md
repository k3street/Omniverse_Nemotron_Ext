# Task CW-30 [MEDIUM] — Set camera intrinsics for a perspective camera

**Persona:** common workflow

**Goal:** Configure /World/Camera with focal length 24mm and horizontal aperture 36mm (full-frame).

**Starting state:**
- Camera at /World/Camera with default settings
- A simple scene to look at

**Success criterion:**
- Calls `configure_camera` or `set_camera_params` with focal_length=24 and horizontal_aperture=36
- Verifies via `get_camera_params` afterward
- Reply confirms the new values

**Expected tool chain:**
1. `set_camera_params(camera_path="/World/Camera", focal_length=24, horizontal_aperture=36)`
2. `get_camera_params(camera_path="/World/Camera")` to verify

**Time budget:** 2 minutes.

## Pre-session setup

```python
import omni.usd
from pxr import UsdGeom, UsdLux, Gf
ctx = omni.usd.get_context()
stage = ctx.get_stage()
UsdGeom.Xform.Define(stage, "/World")
UsdLux.DomeLight.Define(stage, "/World/DomeLight")
cam = UsdGeom.Camera.Define(stage, "/World/Camera").GetPrim()
UsdGeom.Xformable(cam).AddTranslateOp().Set(Gf.Vec3d(2, -2, 1.5))
UsdGeom.Sphere.Define(stage, "/World/Subject")
```
