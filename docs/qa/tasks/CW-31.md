# Task CW-31 [EASY] — Get the world transform of a prim

**Persona:** common workflow

**Goal:** Read and report the world position and orientation of `/World/Cube`.

**Starting state:**
- A cube exists at `/World/Cube` translated to (1, 2, 0.5)

**Success criterion:**
- Calls `get_world_transform(prim_path="/World/Cube")`
- Reply names a position roughly equal to (1, 2, 0.5)

**Expected tool chain:**
1. `get_world_transform(prim_path="/World/Cube")`

**Time budget:** 1 minute.

## Pre-session setup

```python
import omni.usd
from pxr import UsdGeom, UsdLux, Gf
ctx = omni.usd.get_context()
stage = ctx.get_stage()
UsdGeom.Xform.Define(stage, "/World")
UsdLux.DomeLight.Define(stage, "/World/DomeLight")
cube = UsdGeom.Cube.Define(stage, "/World/Cube").GetPrim()
UsdGeom.Xformable(cube).AddTranslateOp().Set(Gf.Vec3d(1, 2, 0.5))
```
