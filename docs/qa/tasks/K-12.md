# Task K-12 [HARD] — Kenji: Fixture alignment on a linear rail

**Persona:** `03_kenji` (Kenji Matsumoto — manufacturing engineer)

**Goal:** Kenji has 3 fixture prims at `/World/Cell/fixture_a/b/c` that need to be aligned on a linear rail at `x = 0.0, 0.2, 0.4` (10mm tolerance), with `y = z = 0`. They currently sit at arbitrary positions. Agent must move each via `teleport_prim` and verify the final positions via `get_world_transform`.

**Starting state:**
- `/World/Cell` (Xform parent)
- `/World/Cell/fixture_a` at (1.0, 0.5, 0.3)
- `/World/Cell/fixture_b` at (0.7, 0.1, -0.2)
- `/World/Cell/fixture_c` at (-0.5, 0.3, 0.8)
- Each is a plain Cube prim (no physics)

**Success criterion (snapshot-measurable):**
- After agent's work, snapshot `world_translations` shows:
  - `/World/Cell/fixture_a` at `(0.0, 0.0, 0.0) ± 0.01`
  - `/World/Cell/fixture_b` at `(0.2, 0.0, 0.0) ± 0.01`
  - `/World/Cell/fixture_c` at `(0.4, 0.0, 0.0) ± 0.01`
- Reply confirms the three final positions (pose claims, which the orchestrator's verify-contract (c) will cross-check against `get_world_transform`).
- `fabricated_claims` empty.

**Expected tool chain (illustrative):**
1. `get_world_transform` on all three to read initial state
2. `teleport_prim(prim_path=..., position=[x, 0.0, 0.0])` per fixture with x=0.0 / 0.2 / 0.4
3. `get_world_transform` on all three to verify round-trip
4. Reply with the 3 final poses stated explicitly.

**Friction points:**
- 10mm tolerance is strict — if agent uses non-uniform scale or leaves a stray translate op, mis-aligned.
- Existing translate ops must be RESET, not compounded — `_safe_set_translate` handles that, so `teleport_prim` is the right tool (uses the safe snippet). If agent adds a new `AddTranslateOp`, the translate would stack on top of the existing authored one — off by (1.0, 0.5, 0.3) from target.
- Agent might claim all three are aligned without the verification pass — orchestrator's verify-contract (c) will catch the poses if reply states them.

**Time budget:** 3 minutes.

## Pre-session setup

```python
import omni.usd
from pxr import UsdGeom, Gf
ctx = omni.usd.get_context()
stage = ctx.get_stage()
UsdGeom.Xform.Define(stage, "/World")
UsdGeom.Xform.Define(stage, "/World/Cell")
for name, pos in [("fixture_a", (1.0, 0.5, 0.3)),
                  ("fixture_b", (0.7, 0.1, -0.2)),
                  ("fixture_c", (-0.5, 0.3, 0.8))]:
    p = UsdGeom.Cube.Define(stage, f"/World/Cell/{name}")
    UsdGeom.Xformable(p).AddTranslateOp().Set(Gf.Vec3d(*pos))
print("seeded K-12: /World/Cell/fixture_{a,b,c} at misaligned positions")
```
