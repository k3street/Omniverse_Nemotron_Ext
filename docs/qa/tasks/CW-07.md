# Task CW-07 [MEDIUM] — Add proximity sensor near robot end-effector

**Persona:** common workflow

**Goal:** Add a proximity sensor (5cm cube) 10cm in front of `/World/Robot/end_effector`.

**Starting state:**
- Stage with a robot Xform at /World/Robot and a child end_effector Xform
- Default light + physics scene

**Success criterion:**
- A proximity sensor prim is created
- Sensor has PhysxTriggerAPI applied (or via `add_proximity_sensor`)
- Position is approximately 10cm in front of the end_effector
- Reply confirms the sensor path

**Expected tool chain:**
1. `get_world_transform(prim_path="/World/Robot/end_effector")` to find current pose
2. `add_proximity_sensor(sensor_path="/World/PickSensor", position=[...], size=[0.05,0.05,0.05])`

**Time budget:** 3 minutes.

## Pre-session setup

```python
import omni.usd
from pxr import UsdGeom, UsdPhysics, UsdLux, Gf
ctx = omni.usd.get_context()
stage = ctx.get_stage()
UsdGeom.Xform.Define(stage, "/World")
UsdPhysics.Scene.Define(stage, "/World/PhysicsScene")
UsdLux.DomeLight.Define(stage, "/World/DomeLight")

robot = UsdGeom.Xform.Define(stage, "/World/Robot").GetPrim()
UsdPhysics.ArticulationRootAPI.Apply(robot)
UsdGeom.Cube.Define(stage, "/World/Robot/base")
ee = UsdGeom.Xform.Define(stage, "/World/Robot/end_effector").GetPrim()
UsdGeom.Xformable(ee).AddTranslateOp().Set(Gf.Vec3d(0.5, 0, 0.5))
```
