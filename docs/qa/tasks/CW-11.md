# Task CW-11 [EASY] — Read current joint positions of a robot

**Persona:** common workflow

**Goal:** Report the current joint positions of `/World/Robot` as a list of floats.

**Starting state:**
- Real Franka URDF imported at /World/Robot (via cuRobo bundled URDF)
- Default light + physics scene

**Success criterion:**
- Calls `get_joint_positions` or `get_articulation_state` on /World/Robot
- Reply lists the joint positions (or values, even if all zero)
- Does NOT fabricate values — if the tool returns null/empty, say so

**Expected tool chain:**
1. `get_joint_positions(articulation_path="/World/Robot")` OR `get_articulation_state(prim_path="/World/Robot")`
2. Reply with the returned values

**Time budget:** 1 minute.

## Pre-session setup

```python
import omni.usd
from pxr import UsdGeom, UsdPhysics, Gf, UsdLux
ctx = omni.usd.get_context()
stage = ctx.get_stage()
UsdGeom.Xform.Define(stage, "/World")
UsdPhysics.Scene.Define(stage, "/World/PhysicsScene")
UsdLux.DomeLight.Define(stage, "/World/DomeLight")

# Manually-authored Franka-like articulation with panda_joint1
robot = UsdGeom.Xform.Define(stage, "/World/Robot").GetPrim()
UsdPhysics.ArticulationRootAPI.Apply(robot)
UsdPhysics.RigidBodyAPI.Apply(robot)
UsdPhysics.MassAPI.Apply(robot).CreateMassAttr(15.0)

base = UsdGeom.Cube.Define(stage, "/World/Robot/base").GetPrim()
UsdGeom.Xformable(base).AddScaleOp().Set(Gf.Vec3d(0.1, 0.1, 0.05))
UsdPhysics.RigidBodyAPI.Apply(base)
UsdPhysics.CollisionAPI.Apply(base)

link1 = UsdGeom.Cube.Define(stage, "/World/Robot/link1").GetPrim()
UsdGeom.Xformable(link1).AddTranslateOp().Set(Gf.Vec3d(0, 0, 0.2))
UsdGeom.Xformable(link1).AddScaleOp().Set(Gf.Vec3d(0.05, 0.05, 0.15))
UsdPhysics.RigidBodyAPI.Apply(link1)
UsdPhysics.CollisionAPI.Apply(link1)

# panda_joint1 — RevoluteJoint connecting base → link1, axis Z
joint1 = UsdPhysics.RevoluteJoint.Define(stage, "/World/Robot/panda_joint1")
joint1.CreateBody0Rel().SetTargets(["/World/Robot/base"])
joint1.CreateBody1Rel().SetTargets(["/World/Robot/link1"])
joint1.CreateAxisAttr("Z")
joint1.CreateLowerLimitAttr(-2.8973)
joint1.CreateUpperLimitAttr(2.8973)
# Drive API for set_joint_targets to work
drive = UsdPhysics.DriveAPI.Apply(joint1.GetPrim(), "angular")
drive.CreateTypeAttr("force")
drive.CreateMaxForceAttr(50.0)
drive.CreateDampingAttr(2.0)
drive.CreateStiffnessAttr(100.0)

```
