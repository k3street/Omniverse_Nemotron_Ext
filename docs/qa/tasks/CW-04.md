**Persona:** common workflow

**Goal:** Import the Franka Panda URDF from the local fixture and report how many joints the imported articulation has.

- Empty stage
- Local URDF at `~/robots/franka_panda/franka_panda.urdf` (or `panda.urdf`)

**Success criterion:**
- Discovers the URDF via `list_local_files` or `catalog_search` if path not in prompt
- Calls `import_robot` (or equivalent) to bring the Franka into the stage at `/World/Franka`

1. `list_local_files(pattern="*franka*", extensions=[".urdf"])`
2. `import_robot(file_path=<found path>, dest_path="/World/Franka", format="urdf")`
3. `verify_import` or `get_articulation_state` to count joints
4. Reply with joint count

**Time budget:** 5 minutes.
