# Task CW-02 [EASY] — Add a dome light to the scene

**Persona:** common workflow

**Goal:** The scene is dark. Add a DomeLight at the default location with intensity 1500.

**Starting state:**
- Stage with a single cube but no lights

**Success criterion:**
- A UsdLux.DomeLight prim is added to the stage
- Its intensity is set to 1500 (within ±50)
- Reply confirms the light was created with its prim path

**Expected tool chain:**
1. `create_prim(prim_type="DomeLight", ...)` OR `add_default_light(...)`

**Time budget:** 1 minute.

## Pre-session setup

```python
import omni.usd
from pxr import UsdGeom
ctx = omni.usd.get_context()
stage = ctx.get_stage()
UsdGeom.Xform.Define(stage, "/World")
UsdGeom.Cube.Define(stage, "/World/TargetCube")
```
