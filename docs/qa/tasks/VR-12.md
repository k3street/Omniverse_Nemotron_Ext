# Task VR-12 [HARD] — Quantitative: 3 cubes 0.5m apart

**Persona:** common workflow

**Goal:** Place three cubes in a row, each 0.5m apart along the X axis.

**Starting state:**
- Empty stage with default light

**Success criterion:**
- Three Cube prims exist
- The pairwise X-distance between consecutive cubes (sorted by X) is 0.5m ± 5cm
- All three cubes have the same Y and Z (within 1cm)
- Reply confirms the row

**Expected tool chain:**
1. `resolve_count_vagueness(term="three")` → 3 (optional — can hardcode)
2. `resolve_constraint_phrase(phrase="0.5m apart")` → parsed.value = 0.5
3. `create_prim` ×3 at positions x=0, x=0.5, x=1.0 (or any 0.5-apart triple)

**Quantitative-fulfillment probe.** Even with the spacing constraint
parsed, the agent must compute and place each cube at the right
coordinates. Failure mode: agent places at random or incorrect spacing.

**Time budget:** 2 minutes.

from pxr import UsdGeom, UsdLux
