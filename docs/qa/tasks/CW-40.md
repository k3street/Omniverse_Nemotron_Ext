# Task CW-40 [EASY] — Count prims under a path

**Persona:** common workflow

**Goal:** Report the number of prims under `/World/Group`.

**Starting state:**
- 5 cubes at `/World/Group/Cube_1` .. `/World/Group/Cube_5`

**Success criterion:**
- Calls `count_prims_under_path(path="/World/Group")` or `list_all_prims(...)` and reports the count
- Reply contains the number 5 (or 6 if /World/Group itself is counted)

**Expected tool chain:**
1. `count_prims_under_path(path="/World/Group")`

**Time budget:** 1 minute.

## Pre-session setup

```python
import omni.usd
from pxr import UsdGeom, UsdLux
ctx = omni.usd.get_context()
stage = ctx.get_stage()
UsdGeom.Xform.Define(stage, "/World")
UsdLux.DomeLight.Define(stage, "/World/DomeLight")
UsdGeom.Xform.Define(stage, "/World/Group")
for i in range(1, 6):
    UsdGeom.Cube.Define(stage, f"/World/Group/Cube_{i}")
```
