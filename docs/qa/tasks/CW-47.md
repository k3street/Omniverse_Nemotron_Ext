# Task CW-47 [EASY] — Get camera parameters

**Persona:** common workflow

**Goal:** Read and report the focal length and aperture of `/World/Camera`.

**Starting state:**
- Camera at `/World/Camera` with default settings

**Success criterion:**
- Calls `get_camera_params(camera_path="/World/Camera")`
- Reply contains numeric focal length and aperture values

**Expected tool chain:**
1. `get_camera_params(camera_path="/World/Camera")`

**Time budget:** 1 minute.

## Pre-session setup

```python
import omni.usd
from pxr import UsdGeom, UsdLux, Gf
ctx = omni.usd.get_context()
stage = ctx.get_stage()
UsdGeom.Xform.Define(stage, "/World")
UsdLux.DomeLight.Define(stage, "/World/DomeLight")
cam = UsdGeom.Camera.Define(stage, "/World/Camera").GetPrim()
UsdGeom.Xformable(cam).AddTranslateOp().Set(Gf.Vec3d(2, -2, 1.5))
```
