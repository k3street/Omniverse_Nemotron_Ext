# Task CW-03 [EASY] — Stack 3 cubes vertically

**Persona:** common workflow

**Goal:** Stack three 0.2m cubes vertically along Z, no gap, bottom cube on the ground plane.

**Starting state:**
- Empty stage with a default light

**Success criterion:**
- Three Cube prims exist
- Cubes are stacked on Z with bottom at z=0.1, middle at z=0.3, top at z=0.5 (centers)
- No air gap between them

**Expected tool chain:**
1. Three `create_prim(prim_type="Cube", position=[0,0,Z], scale=[0.1,0.1,0.1])` calls

**Time budget:** 2 minutes.

## Pre-session setup

```python
import omni.usd
from pxr import UsdGeom, UsdLux
ctx = omni.usd.get_context()
stage = ctx.get_stage()
UsdGeom.Xform.Define(stage, "/World")
UsdLux.DomeLight.Define(stage, "/World/DomeLight")
```
