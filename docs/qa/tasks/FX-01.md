**Time budget:** 5 minutes.

## Pre-session setup

```python
import omni.usd
from pxr import UsdGeom, UsdPhysics, Gf
ctx = omni.usd.get_context()
stage = ctx.get_stage()
UsdGeom.Xform.Define(stage, "/World")
UsdPhysics.Scene.Define(stage, "/World/PhysicsScene")
root = UsdGeom.Xform.Define(stage, "/World/Franka")
UsdPhysics.ArticulationRootAPI.Apply(root.GetPrim())
UsdPhysics.RigidBodyAPI.Apply(root.GetPrim())
base = UsdGeom.Cube.Define(stage, "/World/Franka/base")
base.GetSizeAttr().Set(0.2)
UsdPhysics.RigidBodyAPI.Apply(base.GetPrim())
UsdPhysics.CollisionAPI.Apply(base.GetPrim())
mass = UsdPhysics.MassAPI.Apply(base.GetPrim())
mass.GetMassAttr().Set(5.0)
UsdGeom.Xformable(root).AddTranslateOp().Set(Gf.Vec3d(0.0, 0.0, 0.1))
print("seeded FX-01 unanchored Franka-like articulation at /World/Franka")
```
