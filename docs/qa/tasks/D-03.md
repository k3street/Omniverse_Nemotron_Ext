# Task D-03 [EASY] — Dimitri: List all cameras currently in the scene

**Persona:** `12_dimitri` (Dimitri — lidar/sensor specialist)

**Goal:** Dimitri inherited a warehouse scene from a colleague and needs a quick inventory of every camera prim (mono, stereo, depth) before deciding what to keep.

**Starting state:**
- Scene loaded, ~15 sensors scattered across the stage
- Dimitri suspects some are duplicates and some have wrong intrinsics
- Wants a flat list, not a GUI hunt

**Success criterion:**
- Uses `list_cameras` to enumerate all camera prims
- Returns: prim path, type (RGB / depth / stereo), resolution, frame rate if set
- Flags any camera with suspicious settings (0x0 res, 0 Hz, no parent xform)
- Does NOT just say "check the stage panel"

**Expected tool chain (illustrative):**
1. `list_cameras`
2. `inspect_camera` for anything suspicious found in step 1

**Friction points:**
- Inject: "are any of these RealSense D435 specifically?"
- Inject: "can you group them by the prim they're mounted on?"
- Adversarial: does it list sensors that aren't cameras (e.g. lidar, IMU)?

**Time budget:** 10 minutes.
