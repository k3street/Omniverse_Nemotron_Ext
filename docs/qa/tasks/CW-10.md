# Task CW-10 [MEDIUM] — Setup pick-place controller for an existing cell

**Persona:** common workflow

**Goal:** A pick-place cell already exists with Franka, conveyor, 4 cubes, and a bin. Wire up the pick-place controller to deliver the cubes from belt to bin using the spline target source.

**Starting state:**
- /World/Franka (articulation)
- /World/ConveyorBelt with surface velocity
- /World/Cube_1..4 on the belt
- /World/Bin
- /World/PickSensor
- Default light + physics scene

**Success criterion:**
- Calls `setup_pick_place_controller` with target_source='spline' (NOT 'auto')
- Provides robot_path, sensor_path, belt_path, source_paths (list of cubes), destination_path
- Reply confirms the controller is registered and ready

**Expected tool chain:**
1. `scene_summary` to confirm all components present
2. `setup_pick_place_controller(robot_path="/World/Franka", target_source="spline", sensor_path="/World/PickSensor", belt_path="/World/ConveyorBelt", source_paths=["/World/Cube_1","/World/Cube_2","/World/Cube_3","/World/Cube_4"], destination_path="/World/Bin")`

**Time budget:** 3 minutes.

## Pre-session setup

```python
import omni.usd
from pxr import UsdGeom, UsdPhysics, UsdLux, Gf, Sdf
ctx = omni.usd.get_context()
stage = ctx.get_stage()
UsdGeom.Xform.Define(stage, "/World")
UsdPhysics.Scene.Define(stage, "/World/PhysicsScene")
UsdLux.DomeLight.Define(stage, "/World/DomeLight")

# Table at z=0.75
table = UsdGeom.Cube.Define(stage, "/World/Table").GetPrim()
UsdGeom.Xformable(table).AddTranslateOp().Set(Gf.Vec3d(0, 0, 0.375))
UsdGeom.Xformable(table).AddScaleOp().Set(Gf.Vec3d(1, 0.5, 0.375))
UsdPhysics.CollisionAPI.Apply(table)

# Franka placeholder articulation
franka = UsdGeom.Xform.Define(stage, "/World/Franka").GetPrim()
UsdPhysics.ArticulationRootAPI.Apply(franka)
UsdGeom.Xformable(franka).AddTranslateOp().Set(Gf.Vec3d(0, 0, 0.75))
UsdGeom.Cube.Define(stage, "/World/Franka/base")

# Conveyor at y=0.3
belt = UsdGeom.Cube.Define(stage, "/World/ConveyorBelt").GetPrim()
UsdGeom.Xformable(belt).AddTranslateOp().Set(Gf.Vec3d(0, 0.3, 0.8))
UsdGeom.Xformable(belt).AddScaleOp().Set(Gf.Vec3d(0.8, 0.15, 0.05))
UsdPhysics.CollisionAPI.Apply(belt)

# 4 cubes on belt
for i, x in enumerate([-0.6, -0.4, -0.2, 0.0], 1):
    p = UsdGeom.Cube.Define(stage, f"/World/Cube_{i}").GetPrim()
    UsdGeom.Xformable(p).AddTranslateOp().Set(Gf.Vec3d(x, 0.3, 0.875))
    UsdGeom.Xformable(p).AddScaleOp().Set(Gf.Vec3d(0.025, 0.025, 0.025))
    UsdPhysics.RigidBodyAPI.Apply(p)
    UsdPhysics.CollisionAPI.Apply(p)

# Bin at y=-0.4
bin_p = UsdGeom.Cube.Define(stage, "/World/Bin").GetPrim()
UsdGeom.Xformable(bin_p).AddTranslateOp().Set(Gf.Vec3d(0, -0.4, 0.825))
UsdGeom.Xformable(bin_p).AddScaleOp().Set(Gf.Vec3d(0.15, 0.15, 0.075))

# Proximity sensor
sensor = UsdGeom.Cube.Define(stage, "/World/PickSensor").GetPrim()
UsdGeom.Xformable(sensor).AddTranslateOp().Set(Gf.Vec3d(0.3, 0.3, 0.86))
UsdGeom.Xformable(sensor).AddScaleOp().Set(Gf.Vec3d(0.02, 0.02, 0.02))
```
