## Pre-session setup

```python
import omni.usd
from pxr import UsdGeom, UsdPhysics, Gf
ctx = omni.usd.get_context()
stage = ctx.get_stage()
UsdGeom.Xform.Define(stage, "/World")
UsdPhysics.Scene.Define(stage, "/World/PhysicsScene")

# Robot with end-effector — Yuki points the camera at the EE
robot = UsdGeom.Xform.Define(stage, "/World/Robot").GetPrim()
UsdPhysics.ArticulationRootAPI.Apply(robot)
UsdGeom.Cube.Define(stage, "/World/Robot/base")
ee = UsdGeom.Xform.Define(stage, "/World/Robot/end_effector").GetPrim()
UsdGeom.Xformable(ee).AddTranslateOp().Set(Gf.Vec3d(0.5, 0, 0.5))
tool = UsdGeom.Cube.Define(stage, "/World/Robot/end_effector/tool").GetPrim()
UsdGeom.Xformable(tool).AddScaleOp().Set(Gf.Vec3d(0.04, 0.04, 0.08))
```
