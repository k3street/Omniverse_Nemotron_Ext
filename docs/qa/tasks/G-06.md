# Task G-06 [EASY] — Alex: Move an existing cube to a new absolute position

**Persona:** `08_alex` (Alex — indie developer)

**Goal:** A cube exists at `/World/Box` at position (1, 0, 0). Move it to world position (3, 2, 0.5). Do not create a new cube — move the existing one.

**Starting state:**
- Pre-seeded: cube at `/World/Box`, size=0.4, at translate=(1, 0, 0)

**Success criterion (snapshot-measurable):**
- Exactly one cube named `/World/Box` (same path, not renamed or duplicated)
- `world_translation` ≈ (3, 2, 0.5) within 0.05 m
- Size unchanged (≈0.4)

**Expected tool chain (illustrative):**
1. `prim_exists(prim_path="/World/Box")` to confirm target
2. `get_world_transform` to read current state (optional, confirms pre-seed)
3. `set_attribute(prim_path="/World/Box", attr_name="xformOp:translate", value=[3, 2, 0.5])` OR a dedicated move helper
4. `get_world_transform` to verify

**Friction points:**
- Adversarial: adding a new `AddTranslateOp` on a prim that already has one — breaks xformOpOrder.
- Inject: "make a copy at the new spot" — refuse: task says move, not duplicate.

**Time budget:** 3 minutes.

## Pre-session setup

```python
import omni.usd
from pxr import UsdGeom, Gf
ctx = omni.usd.get_context()
stage = ctx.get_stage()
UsdGeom.Xform.Define(stage, "/World")
c = UsdGeom.Cube.Define(stage, "/World/Box")
c.GetSizeAttr().Set(0.4)
UsdGeom.Xformable(c).AddTranslateOp().Set(Gf.Vec3d(1.0, 0.0, 0.0))
print("seeded G-06 /World/Box at (1,0,0)")
```
