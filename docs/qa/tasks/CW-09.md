# Task CW-09 [MEDIUM] — Add a camera and aim it at the robot

**Persona:** common workflow

**Goal:** Create a perspective camera at `/World/HeroCamera` positioned at (3, -3, 2) and aimed at /World/Robot.

**Starting state:**
- Stage with /World/Robot Xform (placeholder cube body)
- Default light + ground plane

**Success criterion:**
- A UsdGeom.Camera prim is created at /World/HeroCamera
- Camera position is (3, -3, 2) ±0.1
- Camera is oriented to look at /World/Robot (verify via `set_camera_look_at` or matching rotation)
- Reply mentions the camera path

**Expected tool chain:**
1. `create_prim(prim_type="Camera", prim_path="/World/HeroCamera", position=[3,-3,2])`
2. `set_camera_look_at(camera_path="/World/HeroCamera", target_path="/World/Robot")`

**Time budget:** 3 minutes.

## Pre-session setup

```python
import omni.usd
from pxr import UsdGeom, UsdPhysics, UsdLux, Gf
ctx = omni.usd.get_context()
stage = ctx.get_stage()
UsdGeom.Xform.Define(stage, "/World")
UsdLux.DomeLight.Define(stage, "/World/DomeLight")

# Ground
ground = UsdGeom.Cube.Define(stage, "/World/Ground").GetPrim()
UsdGeom.Xformable(ground).AddTranslateOp().Set(Gf.Vec3d(0, 0, -0.05))
UsdGeom.Xformable(ground).AddScaleOp().Set(Gf.Vec3d(2, 2, 0.05))
UsdPhysics.CollisionAPI.Apply(ground)

# Robot placeholder
robot = UsdGeom.Xform.Define(stage, "/World/Robot").GetPrim()
UsdPhysics.ArticulationRootAPI.Apply(robot)
body = UsdGeom.Cube.Define(stage, "/World/Robot/body").GetPrim()
UsdGeom.Xformable(body).AddScaleOp().Set(Gf.Vec3d(0.2, 0.2, 0.5))
UsdGeom.Xformable(body).AddTranslateOp().Set(Gf.Vec3d(0, 0, 0.5))
```
