# Task CW-15 [EASY] — Place 4 cubes in a 2×2 grid

**Persona:** common workflow

**Goal:** Place four 0.1m cubes in a 2×2 grid on the XY plane with 0.3m spacing, all at z=0.05.

**Starting state:**
- Empty stage with default light

**Success criterion:**
- Four Cube prims exist arranged in a 2×2 grid pattern (any reasonable spacing 0.2-0.5m apart) on the XY plane, all at the same Z height
- Each is approximately 0.1m × 0.1m × 0.1m
- Reply lists all 4 prim paths

**Expected tool chain:**
1. Four `create_prim(prim_type="Cube", ...)` calls

**Time budget:** 2 minutes.

## Pre-session setup

```python
import omni.usd
from pxr import UsdGeom, UsdLux
ctx = omni.usd.get_context()
stage = ctx.get_stage()
UsdGeom.Xform.Define(stage, "/World")
UsdLux.DomeLight.Define(stage, "/World/DomeLight")
```
