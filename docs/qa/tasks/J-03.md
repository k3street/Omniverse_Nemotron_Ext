# Task J-03 [MEDIUM] — Jordan: Explain what's in a loaded scene for a student who asks

**Persona:** `11_jordan` (Jordan — educator)

**Goal:** A student just loaded the warehouse template and asked "what am I looking at, what are all these prims?" Jordan needs to turn the raw stage structure into a pedagogic explanation — prim-by-prim, at the level of a second-year undergrad.

**Starting state:**
- Warehouse scene template loaded
- Student is staring at the stage panel with ~80 prims
- Jordan is walking around the classroom, can't sit and type for long

**Success criterion:**
- Uses `run_stage_analysis` to enumerate the stage
- Returns a grouped summary (Environment, Robots, Sensors, Lights) — not the raw list
- Explains at least one example from each group in one sentence ("this is the floor plane, flat ground for the robot to drive on")
- Flags if there's anything unusual / worth pointing out as a teaching moment
- Does NOT produce a 200-line dump with no narrative

**Expected tool chain (illustrative):**
1. `run_stage_analysis`
2. `list_scene_templates` only if student needs to pick a different one

**Friction points:**
- Inject: "the student also wants to know which prim is the one the robot drives — can you point it out?"
- Inject: "is this the same layout the book uses?"
- Adversarial: does it dump every prim path without structure?

**Time budget:** 15 minutes.

## Pre-session setup

```python
import omni.usd
from pxr import UsdGeom, UsdPhysics, UsdLux, Gf
ctx = omni.usd.get_context()
stage = ctx.get_stage()
UsdGeom.Xform.Define(stage, "/World")
UsdPhysics.Scene.Define(stage, "/World/PhysicsScene")

# Lights group
UsdGeom.Xform.Define(stage, "/World/Lights")
UsdLux.DomeLight.Define(stage, "/World/Lights/DomeLight")
UsdLux.RectLight.Define(stage, "/World/Lights/CeilingLight_A")
UsdLux.RectLight.Define(stage, "/World/Lights/CeilingLight_B")

# Environment group: floor, walls, racks, conveyor, dock door
UsdGeom.Xform.Define(stage, "/World/Environment")
floor = UsdGeom.Cube.Define(stage, "/World/Environment/Floor").GetPrim()
UsdGeom.Xformable(floor).AddScaleOp().Set(Gf.Vec3d(15, 15, 0.05))
UsdGeom.Xformable(floor).AddTranslateOp().Set(Gf.Vec3d(0, 0, -0.5))
for i in range(4):
    w = UsdGeom.Cube.Define(stage, f"/World/Environment/Wall_{i}").GetPrim()
for i in range(8):
    UsdGeom.Cube.Define(stage, f"/World/Environment/Rack_{i:02d}")
for i in range(4):
    UsdGeom.Cube.Define(stage, f"/World/Environment/Conveyor_{i}")
UsdGeom.Cube.Define(stage, "/World/Environment/DockDoor")

# Robots group: 2 placeholder AMRs
UsdGeom.Xform.Define(stage, "/World/Robots")
for i in range(2):
    r = UsdGeom.Xform.Define(stage, f"/World/Robots/AMR_{i}").GetPrim()
    UsdPhysics.ArticulationRootAPI.Apply(r)
    UsdGeom.Cube.Define(stage, f"/World/Robots/AMR_{i}/chassis")
    UsdGeom.Xform.Define(stage, f"/World/Robots/AMR_{i}/wheel_FL")
    UsdGeom.Xform.Define(stage, f"/World/Robots/AMR_{i}/wheel_FR")

# Sensors group: 3 cameras + 2 lidar Xforms
UsdGeom.Xform.Define(stage, "/World/Sensors")
for i in range(3):
    UsdGeom.Camera.Define(stage, f"/World/Sensors/Camera_{i}")
for i in range(2):
    UsdGeom.Xform.Define(stage, f"/World/Sensors/Lidar_{i}")
```
