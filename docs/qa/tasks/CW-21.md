# Task CW-21 [MEDIUM] — Apply an impulse force to a rigid body

**Persona:** common workflow

**Goal:** Apply a 5 N upward force to /World/Cube to push it into the air.

**Starting state:**
- Cube at /World/Cube with RigidBodyAPI + CollisionAPI
- Ground plane below
- Default light + physics scene

**Success criterion:**
- Calls `apply_force(prim_path="/World/Cube", force=[0, 0, 5])` or similar
- Reply confirms the force was applied
- Does NOT just describe — uses the tool

**Expected tool chain:**
1. `apply_force(prim_path="/World/Cube", force=[0, 0, 5])`

**Time budget:** 2 minutes.

## Pre-session setup

```python
import omni.usd
from pxr import UsdGeom, UsdPhysics, UsdLux, Gf
ctx = omni.usd.get_context()
stage = ctx.get_stage()
UsdGeom.Xform.Define(stage, "/World")
UsdPhysics.Scene.Define(stage, "/World/PhysicsScene")
UsdLux.DomeLight.Define(stage, "/World/DomeLight")
ground = UsdGeom.Cube.Define(stage, "/World/Ground").GetPrim()
UsdGeom.Xformable(ground).AddTranslateOp().Set(Gf.Vec3d(0, 0, -0.05))
UsdGeom.Xformable(ground).AddScaleOp().Set(Gf.Vec3d(2, 2, 0.05))
UsdPhysics.CollisionAPI.Apply(ground)
cube = UsdGeom.Cube.Define(stage, "/World/Cube").GetPrim()
UsdGeom.Xformable(cube).AddTranslateOp().Set(Gf.Vec3d(0, 0, 0.5))
UsdGeom.Xformable(cube).AddScaleOp().Set(Gf.Vec3d(0.05, 0.05, 0.05))
UsdPhysics.RigidBodyAPI.Apply(cube)
UsdPhysics.CollisionAPI.Apply(cube)
UsdPhysics.MassAPI.Apply(cube).CreateMassAttr(0.5)
```
