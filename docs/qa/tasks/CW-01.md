# Task CW-01 [EASY] — Place a cube at exact coordinates

**Persona:** common workflow

**Goal:** Place a 0.2m cube at world coordinates (1, 2, 0.5).

**Starting state:**
- Empty stage with a default light

**Success criterion:**
- A Cube prim exists at the requested position (within 1mm)
- The cube has size approximately 0.2m × 0.2m × 0.2m
- The reply names the prim path that was created

**Expected tool chain:**
1. `create_prim(prim_type="Cube", position=[1, 2, 0.5], scale=[0.1, 0.1, 0.1])` (USD Cube default size=2 → scale 0.1 = 0.2m)

**Time budget:** 1 minute.

## Pre-session setup

```python
import omni.usd
from pxr import UsdGeom, UsdLux
ctx = omni.usd.get_context()
stage = ctx.get_stage()
UsdGeom.Xform.Define(stage, "/World")
UsdLux.DomeLight.Define(stage, "/World/DomeLight")
```
