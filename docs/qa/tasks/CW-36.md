# Task CW-36 [EASY] — Hide a prim from the viewport

**Persona:** common workflow

**Goal:** Make `/World/Cube` invisible (without deleting it).

**Starting state:**
- A cube at `/World/Cube`

**Success criterion:**
- Calls `set_attribute(prim_path="/World/Cube", attribute_name="visibility", value="invisible")`
  OR a dedicated visibility tool
- Reply confirms cube is hidden

**Expected tool chain:**
1. `set_attribute(prim_path="/World/Cube", attribute_name="visibility", value="invisible")`

**Time budget:** 1 minute.

## Pre-session setup

```python
import omni.usd
from pxr import UsdGeom, UsdLux
ctx = omni.usd.get_context()
stage = ctx.get_stage()
UsdGeom.Xform.Define(stage, "/World")
UsdLux.DomeLight.Define(stage, "/World/DomeLight")
UsdGeom.Cube.Define(stage, "/World/Cube")
```
