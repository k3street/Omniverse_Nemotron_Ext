# Task CW-22 [EASY] — Switch viewport to RTX path-traced rendering

**Persona:** common workflow

**Goal:** Change the renderer to RTX Path Tracer for high-quality look-dev rendering.

**Starting state:**
- A scene with a robot or product
- Currently using default RTX Real-Time

**Success criterion:**
- Calls `set_render_mode(mode="path_traced")` or `set_render_mode(mode="RTX_Interactive")` (the path-traced one)
- Reply confirms mode change

**Expected tool chain:**
1. `set_render_mode(mode="path_traced")` OR equivalent

**Time budget:** 1 minute.

## Pre-session setup

```python
import omni.usd
from pxr import UsdGeom, UsdLux
ctx = omni.usd.get_context()
stage = ctx.get_stage()
UsdGeom.Xform.Define(stage, "/World")
UsdLux.DomeLight.Define(stage, "/World/DomeLight")
UsdGeom.Sphere.Define(stage, "/World/Sphere")
```
