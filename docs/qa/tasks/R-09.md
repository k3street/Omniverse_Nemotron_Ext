**Persona:** `10_raj` (Raj — freelance roboticist, deadline-driven)

**Goal:** Raj took on a quick contract: two robot arms cooperating on pick-and-place. He does NOT want to build from scratch — there has to be a template.

**Starting state:**
- Fresh Isaac Sim session
- Client spec is vague: "two Frankas, one table, hand off a block"
- Raj has 3 other projects running

**Success criterion:**
- Uses `list_scene_templates` (or `list_starter_prompts`) to enumerate what's available
- Picks a reasonable starting template (dual-arm manipulation, or single-arm that can be duplicated)
- Loads it via `load_scene_template`
- If no exact match exists, recommends the closest + the ONE modification needed
- Does NOT recommend "you'll need to build this from scratch"

**Expected tool chain (illustrative):**
1. `list_scene_templates`
2. `load_scene_template` with the chosen name
3. `add_reference` if the template is single-arm and needs a second Franka

**Friction points:**
- Inject: "do you have one with UR10s instead of Frankas?"
- Inject: "can the two arms avoid colliding with each other automatically?"
- Adversarial: does it invent a template name that doesn't exist?

**Time budget:** 10 minutes.
