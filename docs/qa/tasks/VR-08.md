# Task VR-08 [HARD] — Composition: size × material × spatial

**Persona:** common workflow

**Goal:** Create a small rubber cube and a tiny metal sphere on top of it.

**Starting state:**
- Empty stage with default light + physics scene

**Success criterion:**
- A Cube prim exists with size ≈ 0.05m
- A Sphere prim exists with radius ≈ 0.02m, sitting on top of the cube (sphere bottom z >= cube top z, within 5mm tolerance)
- Some physics signal applied to each — RigidBodyAPI / PhysicsMaterialAPI / material reference distinguishing rubber from metal (different friction or density values, or different Material prim references)
- Reply confirms both materials and the stacking

**Expected tool chain:**
1. `resolve_size_adjective(adjective="small", object_class="cube")` → 0.05
2. `resolve_size_adjective(adjective="tiny", object_class="sphere")` → 0.02
3. `resolve_material_properties(material="rubber")` → friction~0.9
4. `resolve_material_properties(material="metal")` → friction~0.6
5. `create_prim` ×2 with sizes from #1, #2
6. Apply materials (`apply_physics_material` / `apply_api_schema`)
7. `place_on_top_of(prim_path="/World/Sphere", target_prim_path="/World/Cube")`

**Time budget:** 3 minutes.

## Pre-session setup

```python
setup_world(physics=True, light=True)
```
