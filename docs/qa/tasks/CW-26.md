# Task CW-26 [EASY] — Find all rigid bodies in the scene

**Persona:** common workflow

**Goal:** List all prim paths that have PhysicsRigidBodyAPI applied.

**Starting state:**
- 3 cubes with RigidBodyAPI, 2 cubes without
- Default light + physics scene

**Success criterion:**
- Calls `find_prims_by_schema(schema_name="PhysicsRigidBodyAPI")`
- Reply enumerates exactly the 3 prim paths with the schema
- Does NOT include the 2 without

**Expected tool chain:**
1. `find_prims_by_schema(schema_name="PhysicsRigidBodyAPI")`

**Time budget:** 1 minute.

## Pre-session setup

```python
import omni.usd
from pxr import UsdGeom, UsdPhysics, UsdLux, Gf
ctx = omni.usd.get_context()
stage = ctx.get_stage()
UsdGeom.Xform.Define(stage, "/World")
UsdPhysics.Scene.Define(stage, "/World/PhysicsScene")
UsdLux.DomeLight.Define(stage, "/World/DomeLight")
for i in range(5):
    p = UsdGeom.Cube.Define(stage, f"/World/Cube_{i}").GetPrim()
    UsdGeom.Xformable(p).AddTranslateOp().Set(Gf.Vec3d(i*0.3, 0, 0.5))
    UsdGeom.Xformable(p).AddScaleOp().Set(Gf.Vec3d(0.05, 0.05, 0.05))
    if i < 3:
        UsdPhysics.RigidBodyAPI.Apply(p)
        UsdPhysics.CollisionAPI.Apply(p)
```
