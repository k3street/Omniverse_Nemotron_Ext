# Task K-06 [EASY] — Kenji: Import a fixture STEP file

**Persona:** `03_kenji` (Kenji Matsumoto — manufacturing engineer, Tier-1 auto)

**Goal:** Kenji has a customer-supplied fixture in STEP format (`fixture_v7.step`, 142 MB). He needs it in the Isaac Sim scene. In Visual Components this is `File → Import → STEP`.

**Starting state:**
- Working robot cell scene
- STEP file on local disk
- No CAD importer configured yet in Isaac Sim
- Kenji expects a single-click import

**Success criterion:**
- Points at the CAD importer extension in Isaac Sim (real extension name — `omni.kit.asset_importer` / CAD Importer) — NOT "just export to URDF from your CAD tool"
- Honestly notes that STEP import goes through a tesselation step and results in mesh prims, NOT parametric CAD features — precision settings matter for large fixtures
- Recommends reviewing the imported bounding box + polycount before trusting it — a 142 MB STEP can produce millions of triangles at default tesselation
- Mentions that collision meshes should be generated separately (convex decomp or simplified geometry) — the rendered mesh is wrong for physics
- Does NOT suggest rebuilding the fixture from primitives — that's not a realistic ask for customer-supplied CAD

**Expected tool chain (illustrative):**
1. `lookup_knowledge` for the Isaac Sim CAD importer
2. `import_robot`-adjacent (or dedicated CAD import tool) for the STEP file
3. `scene_summary` after import to check polycount and bounds
4. `run_stage_analysis` to verify no disconnected or degenerate geometry
5. `set_attribute` / collision approximation flow for the physics mesh

**Friction points (probe these mid-conversation):**
- Inject: "the customer updates this fixture weekly. Can I set up a watch-folder auto-reimport?" — not out of the box; propose a scripted pipeline
- Adversarial: "why doesn't Isaac Sim just read STEP natively like Visual Components?" — tesselation is needed for the USD / render path; explain honestly
- Version probe: "CAD importer ships with 5.x or is it a separate download?"

**Time budget:** 10 minutes.
