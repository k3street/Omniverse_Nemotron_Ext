# Task CW-05 [MEDIUM] — Object falls through floor on Play

**Persona:** common workflow

**Goal:** When sim plays, the cube falls through the ground. Identify why and fix it.

**Starting state:**
- Stage with a cube above a ground-plane Xform with no CollisionAPI applied
- Physics scene exists

**Success criterion:**
- Diagnoses the missing CollisionAPI on the ground plane (via `diagnose_physics_error`, `check_collisions`, `find_prims_by_schema`, or stage inspection)
- Applies the fix: `apply_api_schema(prim_path="/World/Ground", schema_name="PhysicsCollisionAPI")` (or equivalent)
- Verifies the fix by checking the ground prim has the schema afterward
- Reply explains the cause in one sentence and confirms the fix

**Expected tool chain:**
1. `scene_summary` or `list_all_prims`
2. `apply_api_schema(prim_path="/World/Ground", schema_name="PhysicsCollisionAPI")`
3. `get_attribute` or `list_applied_schemas` to verify

**Time budget:** 3 minutes.

## Pre-session setup

```python
import omni.usd
from pxr import UsdGeom, UsdPhysics, UsdLux, Gf
ctx = omni.usd.get_context()
stage = ctx.get_stage()
UsdGeom.Xform.Define(stage, "/World")
UsdPhysics.Scene.Define(stage, "/World/PhysicsScene")
UsdLux.DomeLight.Define(stage, "/World/DomeLight")

# Ground plane WITHOUT collision (the bug)
ground = UsdGeom.Cube.Define(stage, "/World/Ground").GetPrim()
UsdGeom.Xformable(ground).AddTranslateOp().Set(Gf.Vec3d(0, 0, -0.05))
UsdGeom.Xformable(ground).AddScaleOp().Set(Gf.Vec3d(2, 2, 0.05))
# Intentionally no PhysicsCollisionAPI — that's the trap

# Cube above with rigid body
cube = UsdGeom.Cube.Define(stage, "/World/Cube").GetPrim()
UsdGeom.Xformable(cube).AddTranslateOp().Set(Gf.Vec3d(0, 0, 1))
UsdGeom.Xformable(cube).AddScaleOp().Set(Gf.Vec3d(0.1, 0.1, 0.1))
UsdPhysics.RigidBodyAPI.Apply(cube)
UsdPhysics.CollisionAPI.Apply(cube)
```
